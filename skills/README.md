# moduix project skills

Local project skills used by agents in this repository.

## Skills

- `skills/ui-component-workflow/SKILL.md` - rules and workflow for `packages/ui`
- `skills/docs-workflow/SKILL.md` - rules and workflow for `apps/docs`
- `skills/cross-package-sync/SKILL.md` - synchronization contract between UI and docs

## Routing

Use `AGENTS.md` in repo root as the dispatcher for which skill(s) to activate per task.
