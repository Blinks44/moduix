# Kbd

Keyboard shortcut display component.

`Kbd` is a standalone moduix component and does not wrap a Base UI primitive.

## Defaults

| Prop      | Default   | Values                        |
| --------- | --------- | ----------------------------- |
| `variant` | `default` | `default`, `outline`, `ghost` |
| `size`    | `md`      | `sm`, `md`, `lg`              |

For `KbdGroup`:

| Prop   | Default | Values           |
| ------ | ------- | ---------------- |
| `size` | `md`    | `sm`, `md`, `lg` |
