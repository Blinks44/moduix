# Card

`Card` is a standalone moduix component and does not wrap a Base UI primitive.

Use it as a flexible content container with semantic compound slots:
`CardHeader`, `CardBody`, `CardFooter`, `CardTitle`, `CardDescription`, and `CardAction`.
The root supports `variant`, `size`, `interactive`, `className`, a limited `as` prop for common
sectioning elements, and `render` for semantically correct interactive roots such as links or buttons.
`CardTitle` defaults to `h3` and supports `as` to keep page heading levels correct.
