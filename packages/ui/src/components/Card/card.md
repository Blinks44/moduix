# Card

`Card` is a standalone presentational component.

Use it as a thin styled container with composable parts:
`CardHeader`, `CardContent`, `CardFooter`, `CardTitle`, `CardDescription`, and `CardAction`.
The root keeps a minimal API: `className`, native `div` props, and an optional compact `size`.
`CardTitle` defaults to `h3` and supports `as` for heading-level control.
