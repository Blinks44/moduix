# Forgot password

Request a password reset with a single email field and native validation. The block leaves sending
email and displaying the result to your `onSubmit` handler.

## Install

<ShadcnInstall packageName="forgot-password" itemLabel="block" />

The command copies `forgot-password.tsx` and `forgot-password.module.css` into your configured
components directory and installs `@moduix/react` when needed.

## Use in a route

Render `ForgotPassword` in the route that owns password recovery and pass its real submit handler.
Replace the default `/sign-in` link with your application route.
