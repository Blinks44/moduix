# Sign up

Create an account with native validation for name, email, passwords, and terms acceptance. The
block deliberately leaves authentication and matching-password validation to your `onSubmit`
handler.

## Install

<ShadcnInstall packageName="sign-up" itemLabel="block" />

The command copies `sign-up.tsx` and `sign-up.module.css` into your configured components directory
and installs `@moduix/react` when needed.

## Use in a route

Render `SignUp` in the route that owns registration and pass its real submit handler. Replace the
default `/terms`, `/privacy`, and `/sign-in` links with your application routes.
