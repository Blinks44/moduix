## API Reference

Ark UI does not ship a dedicated `tag` primitive. Moduix builds this component with the official
[Ark factory](https://ark-ui.com/docs/guides/composition#the-ark-factory), follows Ark's
[composition](https://ark-ui.com/docs/guides/composition), [styling](https://ark-ui.com/docs/guides/styling), and
[ref](https://ark-ui.com/docs/guides/ref) guidance.

## Choosing the right component

Use `Tag` when the UI is showing **the selected value itself** and that value may need to be
removed, reordered, or managed as part of the current interface state.

- Choose **`Badge`** when the UI is showing short metadata such as status, category, version, or count.
- Choose **`Tag`** when the UI is showing selected values, filters, assignments, or labels that belong to the current state of the interface.

## Anatomy

The root is also exported as `Tag.Root` for namespace consistency with multipart Ark-style
components.

```text
Tag / Tag.Root
├─ Tag.StartElement (optional)
├─ Tag.Label
└─ Tag.EndElement (optional)
   └─ Tag.CloseTrigger (optional)
```

| Part               | Role                                                                               |
| ------------------ | ---------------------------------------------------------------------------------- |
| `Tag` / `Tag.Root` | Root slot. Applies token layout, border, size, variant, and `asChild` composition. |
| `Tag.Label`        | Truncation wrapper for the visible token value.                                    |
| `Tag.StartElement` | Optional leading slot for icons, avatars, or status markers.                       |
| `Tag.EndElement`   | Optional trailing slot for icons or actions.                                       |
| `Tag.CloseTrigger` | Optional button action for dismissing or removing the token.                       |

## Composition

### Recommended composition

Keep token text in `Tag.Label`. Put decorative or supporting elements in `Tag.StartElement` and
`Tag.EndElement`, and put `Tag.CloseTrigger` inside `Tag.EndElement` when the token is removable.

### When to go custom

Use `asChild` when the root itself needs native button or link semantics. The child must be a single
semantic element, and it owns the interaction behavior.

### Default props

| Surface            | Prop         | Default      | Notes                                                                         |
| ------------------ | ------------ | ------------ | ----------------------------------------------------------------------------- |
| `Tag`              | `variant`    | `default`    | `default`, `secondary`, `outline`, `ghost`, `destructive`                     |
| `Tag`              | `size`       | `md`         | `sm`, `md`                                                                    |
| `Tag.CloseTrigger` | `type`       | `button`     | Prevents accidental form submission when not using `asChild`.                 |
| `Tag.CloseTrigger` | `children`   | close icon   | Used only when not using `asChild`; replaced by custom children.              |
| `Tag.CloseTrigger` | `aria-label` | `Remove tag` | Applied only with no custom children, no `aria-labelledby`, and no `asChild`. |

### Behavioral notes

- `Tag` has no owned state, callbacks, keyboard roving, or form behavior.
- `Tag.CloseTrigger` is explicit composition, not a root-level `onClose` prop.
- Built-in variants use the same color semantics as `Badge`; `Tag` differs by part anatomy and close affordance.
- Give each close trigger a specific accessible name when several removable tags appear together.
- `asChild` follows Ark factory composition and requires one child element.
- `Tag.CloseTrigger asChild` requires an explicit semantic child; the default close icon is rendered only by the default button host.
- The default close trigger composes the shared `CloseButton`; existing `--moduix-tag-close-trigger-*`
  variables and the `tag-close-trigger` data slot remain the styling API.

### Styling hooks

- `data-scope="tag"` and `data-part="root" | "label" | "start-element" | "end-element" | "close-trigger"` are emitted on the exported parts.
- `data-slot="tag-root"`, `data-slot="tag-label"`, `data-slot="tag-start-element"`, `data-slot="tag-end-element"`, and `data-slot="tag-close-trigger"` are stable moduix selectors.
- `Tag` writes the selected size and tone to `data-size` and `data-variant`.
- `Tag.CloseTrigger` exposes `data-disabled` for disabled or `aria-disabled` states and inherits the token color.
