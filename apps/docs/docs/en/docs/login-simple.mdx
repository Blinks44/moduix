# Login simple

A compact email-and-password form with native validation, a remember-me control, and clear account
recovery links. It deliberately does not implement authentication: pass `onSubmit` to connect your
own action or API.

## Install

<ShadcnInstall packageName="login-simple" itemLabel="block" />

The command copies `login-simple.tsx` and `login-simple.module.css` into your configured
components directory and installs `@moduix/react` when needed.

## Use in a route

Render `LoginSimple` in the route that owns authentication and pass its real submit handler. Replace
the default `/forgot-password` and `/sign-up` links with your application routes.
