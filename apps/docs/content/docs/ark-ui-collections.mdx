# Ark UI Collections

Collections are the data layer behind Ark list-style components. They are not styled moduix
components, but they are important when you work with `Select`, `Combobox`, `Listbox`, `TreeView`,
or custom list interactions.

Import collection helpers directly from Ark UI:

```tsx
import {
  createGridCollection,
  createListCollection,
  useListCollection,
} from '@ark-ui/react/collection';
```

moduix components accept Ark collections, but moduix does not wrap or re-export the collection
helpers.

## Overview

| Ark feature                                                            | Use it for                                                                                                |
| ---------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| [List Collection](https://ark-ui.com/docs/collections/list-collection) | Array-backed option data with value lookup, next/previous traversal, disabled items, and reorder support. |
| [Tree Collection](https://ark-ui.com/docs/collections/tree-collection) | Hierarchical data such as folders, navigation trees, nested categories, and organization charts.          |
| [Async List](https://ark-ui.com/docs/collections/async-list)           | Loading, filtering, sorting, pagination, request cancellation, and error state for remote list data.      |
| [List Selection](https://ark-ui.com/docs/collections/list-selection)   | Single, multiple, deselectable, and range-style selection state for list collections.                     |

## When To Use Them

Use collections when a component needs stable item identity, keyboard traversal, or value-to-item
conversion. `createListCollection` is the normal path for `Select`, `Combobox`, and `Listbox`
options. `createGridCollection` fits grid-like listbox layouts. Tree and async helpers are also Ark
tooling for more advanced data ownership.

For component-specific usage, start from the relevant moduix page:

- [Select](/docs/select)
- [Combobox](/docs/combobox)
- [Listbox](/docs/listbox)
- [Tree View](/docs/tree-view)
