# Reset password

Let a user choose a new password after they open a verified reset link. Native validation ensures
both fields are present; confirm the passwords match in your `onSubmit` handler.

## Install

<ShadcnInstall packageName="reset-password" itemLabel="block" />

The command copies `reset-password.tsx` and `reset-password.module.css` into your configured
components directory and installs `@moduix/react` when needed.

## Use in a route

Render `ResetPassword` in the route reached from a verified reset link and pass its real submit
handler. Replace the default `/sign-in` link with your application route.
