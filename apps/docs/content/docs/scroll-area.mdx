## Basic

`ScrollArea` is the recommended path for most cases. It renders the root, viewport, content, and a
default vertical scrollbar for you.

## Anatomy

Use the high-level wrapper when the default structure is enough:

```tsx
import { ScrollArea } from 'moduix';

<ScrollArea>{children}</ScrollArea>;
```

Use the low-level parts when you need direct control over the viewport, content, or scrollbars:

```tsx
import {
  ScrollAreaContent,
  ScrollAreaCorner,
  ScrollAreaRoot,
  ScrollAreaScrollbar,
  ScrollAreaThumb,
  ScrollAreaViewport,
} from 'moduix';

<ScrollAreaRoot>
  <ScrollAreaViewport>
    <ScrollAreaContent>{children}</ScrollAreaContent>
  </ScrollAreaViewport>
  <ScrollAreaScrollbar>
    <ScrollAreaThumb />
  </ScrollAreaScrollbar>
</ScrollAreaRoot>;
```

| Part                  | Role                                                         |
| --------------------- | ------------------------------------------------------------ |
| `ScrollArea`          | High-level wrapper with default viewport, content, and bars. |
| `ScrollAreaRoot`      | Root scroll area container.                                  |
| `ScrollAreaViewport`  | Focusable native scroll viewport.                            |
| `ScrollAreaContent`   | Inner content wrapper.                                       |
| `ScrollAreaScrollbar` | One scrollbar track.                                         |
| `ScrollAreaThumb`     | Draggable scrollbar thumb.                                   |
| `ScrollAreaCorner`    | Fills the corner when both scrollbars are visible.           |

## Composition

The wrapper keeps the API intentionally small:

| Prop         | Default    | Values                                            |
| ------------ | ---------- | ------------------------------------------------- |
| `scrollbars` | `vertical` | `vertical`, `horizontal`, `both`, `false`         |
| `fade`       | `false`    | `false`, `true`, `vertical`, `horizontal`, `both` |

`fade={true}` maps to a vertical edge fade, which matches the most common single-column case.

Use `ScrollArea` when you need:

- root sizing, borders, and spacing through `className`
- the default vertical scrollbar
- an optional horizontal scrollbar via `scrollbars`
- a small edge-fade convenience via `fade`

Use the low-level parts when you need:

- custom classes on viewport or content
- manual scrollbar props such as `keepMounted`
- different scrollbar markup or orientation-specific styling
- a custom root data contract such as `data-fade="both"`

## Examples

### Both Scrollbars

Use `scrollbars="both"` when content can overflow on both axes.

<Preview>
  <BothScrollbarsScrollAreaExample />

  <Preview.Code>
    {`
      import { ScrollArea } from "moduix";

      export function BothScrollbarsScrollAreaDemo() {
        return (
          <ScrollArea scrollbars="both" className={styles.sizedRoot}>
            <div className={styles.gridContent}>
              {Array.from({ length: 96 }, (_, index) => (
                <div key={index} className={styles.cell}>
                  {index + 1}
                </div>
              ))}
            </div>
          </ScrollArea>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>

## Custom Composition

Use the exported parts directly when the default wrapper stops being enough.

<Preview>
  <CustomCompositionScrollAreaExample />

  <Preview.Code>
    {`
      import {
        ScrollAreaContent,
        ScrollAreaCorner,
        ScrollAreaRoot,
        ScrollAreaScrollbar,
        ScrollAreaThumb,
        ScrollAreaViewport,
      } from "moduix";

      export function CustomCompositionScrollAreaDemo() {
        return (
          <ScrollAreaRoot
            className={styles.customRoot}
            data-fade="both"
            overflowEdgeThreshold={28}
          >
            <ScrollAreaViewport className={styles.customViewport}>
              <ScrollAreaContent className={styles.customContent}>
                {Array.from({ length: 80 }, (_, index) => (
                  <div key={index} className={styles.customCell}>
                    {index + 1}
                  </div>
                ))}
              </ScrollAreaContent>
            </ScrollAreaViewport>
            <ScrollAreaScrollbar className={styles.customVerticalScrollbar} keepMounted>
              <ScrollAreaThumb className={styles.customVerticalThumb} />
            </ScrollAreaScrollbar>
            <ScrollAreaScrollbar
              orientation="horizontal"
              className={styles.customHorizontalScrollbar}
            >
              <ScrollAreaThumb className={styles.customHorizontalThumb} />
            </ScrollAreaScrollbar>
            <ScrollAreaCorner className={styles.customCorner} />
          </ScrollAreaRoot>
        );
      }
    `}

  </Preview.Code>
</Preview>
