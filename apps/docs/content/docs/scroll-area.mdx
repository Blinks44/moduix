## Basic

`ScrollArea` is the default path. It renders the root, viewport, content, and matching scrollbars
for you.

## Anatomy

For the common case, use the high-level wrapper:

```tsx
import { ScrollArea } from 'moduix';

<ScrollArea>{children}</ScrollArea>;
```

When you need full control over layout or DOM structure, use the low-level parts directly:

```tsx
import {
  ScrollAreaContent,
  ScrollAreaCorner,
  ScrollAreaRoot,
  ScrollAreaScrollbar,
  ScrollAreaThumb,
  ScrollAreaViewport,
} from 'moduix';

<ScrollAreaRoot>
  <ScrollAreaViewport>
    <ScrollAreaContent>{children}</ScrollAreaContent>
  </ScrollAreaViewport>
  <ScrollAreaScrollbar>
    <ScrollAreaThumb />
  </ScrollAreaScrollbar>
  <ScrollAreaCorner />
</ScrollAreaRoot>;
```

## Composition

`ScrollArea` keeps the default API small:

| Prop         | Default    | Values                                            |
| ------------ | ---------- | ------------------------------------------------- |
| `scrollbars` | `vertical` | `vertical`, `horizontal`, `both`, `false`         |
| `fade`       | `false`    | `false`, `true`, `vertical`, `horizontal`, `both` |

Other root props such as `overflowEdgeThreshold` are forwarded directly to Base UI.

Use the high-level wrapper when you only need:

- root sizing and borders through `className`
- a default vertical scrollbar
- an optional horizontal scrollbar with `scrollbars="both"` or `scrollbars="horizontal"`
- a fade mask with `fade`

Use the low-level parts when you need:

- custom viewport or content classes
- manual scrollbar props such as `keepMounted`
- a different DOM structure
- per-part styling beyond CSS variables

## Examples

### Both Scrollbars

Use `scrollbars="both"` when content can overflow on both axes. Make the inner content as wide as
it needs to be.

<Preview>
  <BothScrollbarsScrollAreaExample />

  <Preview.Code>
    {`
      import { ScrollArea } from "moduix";

      export function BothScrollbarsScrollAreaDemo() {
        return (
          <ScrollArea scrollbars="both" className={styles.sizedRoot}>
            <div className={styles.gridContent}>
              {Array.from({ length: 96 }, (_, index) => index + 1).map((cell) => (
                <div key={cell} className={styles.cell}>
                  {cell}
                </div>
              ))}
            </div>
          </ScrollArea>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>

### Gradient Fade

Use `fade` to add a built-in edge fade driven by the scroll position CSS variables from Base UI.

<Preview>
  <GradientFadeScrollAreaExample />

  <Preview.Code>
    {`
      import { ScrollArea } from "moduix";

      export function GradientFadeScrollAreaDemo() {
        return (
          <ScrollArea fade className={styles.sizedRoot}>
            <div className={styles.paddedTextContent}>
              {Array.from({ length: 8 }, (_, index) => ({
                title: \`Section \${index + 1}\`,
                body: "Scrollable content stays readable while the edges fade in and out.",
              })).map((item) => (
                <section key={item.title}>
                  <h3>{item.title}</h3>
                  <p className={styles.paragraph}>{item.body}</p>
                </section>
              ))}
            </div>
          </ScrollArea>
        );
      }
    `}

  </Preview.Code>
</Preview>

### Custom Composition

Use the low-level parts when you need direct control over the viewport, content, or scrollbars.

<Preview>
  <CustomCompositionScrollAreaExample />

  <Preview.Code>
    {`
      import {
        ScrollAreaContent,
        ScrollAreaCorner,
        ScrollAreaRoot,
        ScrollAreaScrollbar,
        ScrollAreaThumb,
        ScrollAreaViewport,
      } from "moduix";

      export function CustomCompositionScrollAreaDemo() {
        return (
          <ScrollAreaRoot
            className={styles.customRoot}
            data-fade="both"
            overflowEdgeThreshold={28}
          >
            <ScrollAreaViewport className={styles.customViewport}>
              <ScrollAreaContent className={styles.customContent}>
                {Array.from({ length: 80 }, (_, index) => index + 1).map((cell) => (
                  <div key={cell} className={styles.customCell}>
                    {cell}
                  </div>
                ))}
              </ScrollAreaContent>
            </ScrollAreaViewport>
            <ScrollAreaScrollbar className={styles.customVerticalScrollbar} keepMounted>
              <ScrollAreaThumb className={styles.customVerticalThumb} />
            </ScrollAreaScrollbar>
            <ScrollAreaScrollbar
              orientation="horizontal"
              className={styles.customHorizontalScrollbar}
            >
              <ScrollAreaThumb className={styles.customHorizontalThumb} />
            </ScrollAreaScrollbar>
            <ScrollAreaCorner className={styles.customCorner} />
          </ScrollAreaRoot>
        );
      }
    `}

  </Preview.Code>
</Preview>
