## Default Path

Use `Toggle` as a single styled button. In most cases you only need children plus Base UI state
props like `defaultPressed` or `pressed`.

`Toggle` stays intentionally small: one root, three built-in visual variants, and a compact size
scale for text or icon-only controls.

## Anatomy

```text
Toggle
└─ label and/or icon content
```

| Part     | Role                                                                    |
| -------- | ----------------------------------------------------------------------- |
| `Toggle` | Two-state button root that handles pressed, focus, and disabled states. |

## Composition

`Toggle` is a thin styled wrapper over the Base UI primitive.

Use these paths:

| Pattern                            | Best fit                                                  |
| ---------------------------------- | --------------------------------------------------------- |
| `defaultPressed`                   | Local uncontrolled saved/favorite toggles                 |
| `pressed` + `onPressedChange`      | Controlled state coordinated with other UI                |
| `variant` + `size`                 | Library defaults for common visual treatments             |
| `className`, `style`, `--toggle-*` | Visual overrides without changing structure               |
| `render={(props, state) => ...}`   | Advanced composition when markup or icon depends on state |

### Defaults

| Prop      | Default   | Values                                                  |
| --------- | --------- | ------------------------------------------------------- |
| `variant` | `default` | `default`, `outline`, `ghost`                           |
| `size`    | `md`      | `xs`, `sm`, `md`, `lg`, `icon-sm`, `icon-md`, `icon-lg` |

Icon-only toggles should always include an accessible label such as `aria-label="Favorite"`.

### Controlled

Control `pressed` from React state when the toggle participates in app state or analytics.

<Preview>
  <ControlledToggleExample />

  <Preview.Code>
    {`
      import { BellIcon, Toggle } from "moduix";
      import { useState } from "react";
      import styles from "./toggle.module.css";

      export function ControlledToggleDemo() {
        const [pressed, setPressed] = useState(false);

        return (
          <div className={styles.stack}>
            <Toggle pressed={pressed} onPressedChange={setPressed}>
              <BellIcon />
              {pressed ? "Notifications on" : "Notifications off"}
            </Toggle>
            <span className={styles.hint}>Current value: {String(pressed)}</span>
          </div>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .stack {
        display: flex;
        flex-direction: column;
        align-items: flex-start;

      .hint {
        color: var(--color-muted-foreground);
        font-size: var(--text-xs);
        line-height: var(--line-height-text-xs);
      }
    `}

  </Preview.CSS>
</Preview>

## Custom Composition

Use `render` when the visible element or icon should depend on primitive state. Keep this as the
advanced path; most toggles should stay on normal children.

<Preview>
  <ToggleRenderCallbackExample />

  <Preview.Code>
    {`
      import { Toggle } from "moduix";
      import styles from "./toggle.module.css";

      export function ToggleRenderCallbackDemo() {
        return (
          <Toggle
            aria-label="Save article"
            size="icon-md"
            variant="outline"
            render={(buttonProps, state) => (
              <button type="button" {...buttonProps}>
                {state.pressed ? (
                  <BookmarkFilledIcon className={styles.customIcon} />
                ) : (
                  <BookmarkIcon className={styles.customIcon} />
                )}
              </button>
            )}
          />
        );
      }

      function BookmarkIcon(props) {
        return (
          <svg viewBox="0 0 16 16" fill="none" aria-hidden="true" focusable="false" {...props}>
            <path
              d="M4.5 2.75h7v10.5L8 11l-3.5 2.25V2.75Z"
              stroke="currentColor"
              strokeWidth="1.5"
              strokeLinejoin="round"
            />
          </svg>
        );
      }

      function BookmarkFilledIcon(props) {
        return (
          <svg viewBox="0 0 16 16" fill="currentColor" aria-hidden="true" focusable="false" {...props}>
            <path d="M4 2.75A.75.75 0 0 1 4.75 2h6.5a.75.75 0 0 1 .75.75v10.5a.75.75 0 0 1-1.16.63L8 12.03l-2.84 1.85A.75.75 0 0 1 4 13.25V2.75Z" />
          </svg>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .customIcon {
        width: 1rem;
        height: 1rem;
      }
    `}
  </Preview.CSS>
</Preview>
