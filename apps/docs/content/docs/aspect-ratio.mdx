## API Reference

`AspectRatio` is an Ark-aligned standalone wrapper built with `@ark-ui/react/factory`. Ark UI does
not currently ship a dedicated `aspect-ratio` primitive, so moduix exposes the root as
`AspectRatio.Root` and keeps the wrapper limited to styling, `data-slot`, and `asChild` support.

## Basic

Use `AspectRatio.Root` with a numeric `ratio` for responsive images, videos, embeds, and canvas
content.

## Anatomy

```text
AspectRatio.Root
└─ media | iframe | canvas | custom content
```

| Part               | Role                                  |
| ------------------ | ------------------------------------- |
| `AspectRatio.Root` | Responsive frame with clipping.       |
| `AspectRatio`      | Callable alias of `AspectRatio.Root`. |

## Composition

### Recommended composition

Stay with `AspectRatio.Root` for most media frames.

- `ratio` is required and numeric
- `asChild` is supported through Ark factory props
- direct `img`, `video`, `iframe`, `canvas`, and `svg` children fill the frame automatically
- `className` styles the root directly
- `--aspect-ratio-radius` controls the default rounded frame

### When to go custom

Wrap `AspectRatio.Root` when shadows, focus rings, or layout decoration need to escape the clipped
frame. Keep the clipping on the root itself.

### Default props

| Entry                   | Default            | Values / Notes                    |
| ----------------------- | ------------------ | --------------------------------- |
| `ratio`                 | -                  | Required `number`                 |
| `className`             | `undefined`        | `string`                          |
| `style`                 | `undefined`        | Can override inline `aspectRatio` |
| `asChild`               | `undefined`        | Ark factory composition           |
| `--aspect-ratio-radius` | `var(--radius-md)` | CSS length or token value         |

### Behavioral notes

- The root renders with `data-scope="aspect-ratio"`, `data-part="root"`, and
  `data-slot="aspect-ratio-root"`.
- `position: relative` on the root is intentional and supports absolute-fill child patterns such
  as Next.js `Image fill`.
- The previous preset strings (`square`, `video`, `portrait`, `photo`) were removed during the Ark
  migration. Use numbers such as `1`, `16 / 9`, `9 / 16`, and `4 / 3`.

### Styling hooks

- `data-scope="aspect-ratio"` and `data-part="root"` are the Ark-aligned root hooks.
- `data-slot="aspect-ratio-root"` is the stable moduix root hook.
- `overflow: hidden` is intentional. If visible decoration must escape the frame, put that
  decoration on an outer wrapper instead of removing root clipping.
