## API Reference

`AspectRatio` is a standalone moduix component and does not wrap a Base UI primitive.

## Basic

Start with `ratio="video"` for the most common responsive media frame. This is the recommended
path for images, videos, embeds, and canvas-based content that should keep a stable shape while the
parent width changes.

<Preview>
  <AspectRatioExample />

  <Preview.Code>
    {`
      import { AspectRatio } from "moduix";

      export function AspectRatioDemo() {
        return (
          <div className="aspect-ratio-demo">
            <AspectRatio ratio="video">
              <img
                src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800&q=80"
                alt="Mountain landscape"
                className="aspect-ratio-demo__image"
              />
            </AspectRatio>
          </div>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .aspect-ratio-demo {
        width: min(30rem, calc(100vw - var(--spacing-8)));
      }

      .aspect-ratio-demo__image {
        object-fit: cover;
      }
    `}

  </Preview.CSS>

## Anatomy

`AspectRatio` exposes one root. There are no sub-parts or slot prop systems.

```text
AspectRatio
└─ root[data-slot="aspect-ratio-root"]
```

| Part          | Role                                                                 |
| ------------- | -------------------------------------------------------------------- |
| `AspectRatio` | Root media frame. Controls the ratio, clipping, and rounded surface. |

## Composition

### When to stay high-level

Stay with `AspectRatio` directly for most image, video, iframe, canvas, and SVG frames.

The component stays intentionally small:

- `ratio` is required; there is no default that could hide layout mistakes
- direct `img`, `video`, `iframe`, `canvas`, and `svg` children fill the box automatically
- `className` styles the root directly
- `--aspect-ratio-radius` customizes the default rounded surface without extra wrapper markup

### When to go custom

Switch to a custom wrapper only when something outside the clipped frame needs to escape the box,
such as a shadow, focus treatment, or outer layout decoration. Keep the built-in clipping on
`AspectRatio` itself.

### Default props

| Entry                   | Default            | Values / Notes                                    |
| ----------------------- | ------------------ | ------------------------------------------------- |
| `ratio`                 | -                  | `number`, `square`, `video`, `portrait`, `photo`  |
| `className`             | `undefined`        | `string`                                          |
| `style`                 | `undefined`        | Style object. Can override the inline ratio style |
| `--aspect-ratio-radius` | `var(--radius-md)` | CSS length or token value                         |

### Behavioral notes

- Named presets resolve to numeric ratios at render time: `square`, `video`, `portrait`, `photo`.
- The resolved ratio is written as inline `aspectRatio` CSS.
- Direct media children auto-fill the box, but `object-fit` remains the consumer's responsibility.
- `position: relative` on the root is intentional and supports absolute-fill child patterns such as
  Next.js `<Image fill>`.

### Card Grid

This is the common editorial case: two cards sit side by side, but the uploaded images have very
different source proportions. Giving both media blocks the same `AspectRatio` and `object-fit: cover`
keeps the visual media height aligned instead of letting the cards jump.

<Preview>
  <AspectRatioCardGridExample />

  <Preview.Code>
    {`
      import { AspectRatio, Card } from "moduix";

      export function AspectRatioCardGridDemo() {
        return (
          <div className="aspect-ratio-grid">
            <Card className="aspect-ratio-grid__card">
              <AspectRatio ratio="video" className="aspect-ratio-grid__media">
                <img
                  src="https://images.unsplash.com/photo-1500530855697-b586d89ba3ee?w=1200&h=720&q=80"
                  alt="Evening city skyline"
                  className="aspect-ratio-grid__image"
                />
              </AspectRatio>
              <div className="aspect-ratio-grid__body">
                <h3 className="aspect-ratio-grid__title">City center opens a redesigned pedestrian corridor</h3>
                <p className="aspect-ratio-grid__description">
                  The source image is landscape, but the media frame stays aligned with the card beside it.
                </p>
              </div>
            </Card>

            <Card className="aspect-ratio-grid__card">
              <AspectRatio ratio="video" className="aspect-ratio-grid__media">
                <img
                  src="https://images.unsplash.com/photo-1515886657613-9f3515b0c78f?w=720&h=1200&q=80"
                  alt="Reporter portrait on a street background"
                  className="aspect-ratio-grid__image"
                />
              </AspectRatio>
              <div className="aspect-ratio-grid__body">
                <h3 className="aspect-ratio-grid__title">
                  Editors uploaded a tall portrait photo for this story
                </h3>
                <p className="aspect-ratio-grid__description">
                  AspectRatio plus object-fit: cover keeps the media height the same instead of letting the card jump.
                </p>
              </div>
            </Card>
          </div>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .aspect-ratio-grid {
        display: grid;
        width: min(42rem, calc(100vw - var(--spacing-8)));
        gap: var(--spacing-4);
        align-items: start;

        @media (min-width: 40rem) {

        }
      }

      .aspect-ratio-grid__card {
        overflow: hidden;

      .aspect-ratio-grid__media {
        --aspect-ratio-radius: 0;
        border-end-start-radius: 0;
        border-end-end-radius: 0;
      }

      .aspect-ratio-grid__image {
        object-fit: cover;
      }

      .aspect-ratio-grid__body {
        display: grid;
        gap: var(--spacing-2);
        padding: var(--spacing-5);
      }

      .aspect-ratio-grid__title {
        margin: 0;
        color: var(--color-foreground);
        font-size: var(--text-md);
        font-weight: var(--weight-semibold);
        line-height: var(--line-height-text-md);
      }

## Styling

`AspectRatio` has one public styling token, and its root is the only stable styling surface.

### CSS Properties

### Styling hooks

- `data-slot="aspect-ratio-root"` is the stable selector on the exported root.
- `overflow: hidden` and the default clipping are intentional. If a child needs a visible shadow
  or focus treatment outside the frame, put that treatment on an outer wrapper instead of removing
  root clipping.
