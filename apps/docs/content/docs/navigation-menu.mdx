## Overview

`NavigationMenu` keeps the visible API short, but this primitive still needs a separate popup
viewport behind the scenes. The default root renders that infrastructure for you, so the common
path only needs list, item, trigger, content, and link parts.

## Anatomy

The recommended structure stays close to shadcn-style composition:

```text
NavigationMenu
└─ NavigationMenuList
   ├─ NavigationMenuItem
   │  ├─ NavigationMenuTrigger
   │  └─ NavigationMenuContent
   └─ NavigationMenuItem
      └─ NavigationMenuLink
```

By default, the root also renders:

```text
NavigationMenuPortal
└─ NavigationMenuPositioner
   └─ NavigationMenuPopup
      ├─ NavigationMenuArrow (optional)
      └─ NavigationMenuViewport
```

Those low-level parts stay available when you need a backdrop, a full-width popup, or an inline
viewport.

## Composition

`NavigationMenu` forwards Base UI root props such as `value`, `defaultValue`, `onValueChange`,
`delay`, `closeDelay`, `orientation`, `actionsRef`, `render`, `className`, and `style`.

The root also keeps a narrow popup contract for the built-in viewport composition:

| Prop                                                                                                                                                                         | Default                             | Purpose                                                                                                                        |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `showPopup`                                                                                                                                                                  | `true`                              | Disables the built-in popup composition so you can assemble `Portal`, `Positioner`, `Popup`, `Arrow`, and `Viewport` manually. |
| `showArrow`                                                                                                                                                                  | `false`                             | Turns on the built-in popup arrow.                                                                                             |
| `sideOffset`                                                                                                                                                                 | `10`                                | Default offset for the built-in `Positioner`.                                                                                  |
| `side`, `align`, `alignOffset`, `arrowPadding`, `anchor`, `collisionAvoidance`, `collisionBoundary`, `collisionPadding`, `sticky`, `positionMethod`, `disableAnchorTracking` | Base UI defaults unless noted above | Forwarded to the built-in `Positioner`.                                                                                        |

`NavigationMenuTrigger` keeps one small DX shortcut:

| Prop   | Default                | Purpose                                                      |
| ------ | ---------------------- | ------------------------------------------------------------ |
| `icon` | `ChevronDownSmallIcon` | Replaces the default trigger icon. Pass `null` to remove it. |

When the default path is enough, style `NavigationMenu`, `NavigationMenuTrigger`,
`NavigationMenuContent`, and `NavigationMenuLink`. Switch to explicit parts only when structure or
popup behavior needs to change.

### Nested Inline Submenu

When second-level content should stay inside the current panel, disable the built-in popup and
render `NavigationMenuViewport` inline.

<Preview>
  <NestedInlineNavigationMenuExample />

  <Preview.Code>
    {`
      import {
        NavigationMenu,
        NavigationMenuContent,
        NavigationMenuItem,
        NavigationMenuList,
        NavigationMenuTrigger,
        NavigationMenuViewport,
      } from "moduix";

      export function InlineNavigationMenuDemo() {
        return (
          <NavigationMenu>
            <NavigationMenuList>
              <NavigationMenuItem>
                <NavigationMenuTrigger>Product</NavigationMenuTrigger>
                <NavigationMenuContent>
                  <NavigationMenu
                    defaultValue="developers"
                    orientation="vertical"
                    showPopup={false}
                  >
                    <NavigationMenuList>{/* nested items */}</NavigationMenuList>
                    <NavigationMenuViewport className={styles.inlineViewport} />
                  </NavigationMenu>
                </NavigationMenuContent>
              </NavigationMenuItem>
            </NavigationMenuList>
          </NavigationMenu>
        );
      }
    `}

## Custom Composition

Set `showPopup={false}` when you need to own the popup structure directly.

<Preview>
  <CustomCompositionNavigationMenuExample />

  <Preview.Code>
    {`
      import {
        NavigationMenu,
        NavigationMenuArrow,
        NavigationMenuBackdrop,
        NavigationMenuContent,
        NavigationMenuItem,
        NavigationMenuList,
        NavigationMenuPopup,
        NavigationMenuPortal,
        NavigationMenuPositioner,
        NavigationMenuTrigger,
        NavigationMenuViewport,
      } from "moduix";

      export function CustomNavigationMenuDemo() {
        return (
          <NavigationMenu showPopup={false}>
            <NavigationMenuList>
              <NavigationMenuItem>
                <NavigationMenuTrigger>Resources</NavigationMenuTrigger>
                <NavigationMenuContent>{/* content */}</NavigationMenuContent>
              </NavigationMenuItem>
            </NavigationMenuList>

            <NavigationMenuPortal>
              <NavigationMenuBackdrop />
              <NavigationMenuPositioner sideOffset={16}>
                <NavigationMenuPopup>
                  <NavigationMenuArrow />
                  <NavigationMenuViewport />
                </NavigationMenuPopup>
              </NavigationMenuPositioner>
            </NavigationMenuPortal>
          </NavigationMenu>
        );
      }
    `}

  </Preview.Code>
</Preview>
