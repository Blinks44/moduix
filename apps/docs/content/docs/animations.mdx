# Animations

moduix drives appearance transitions through CSS variables on the visual part that owns the motion.
Ark state attributes trigger the CSS, and Ark presence behavior keeps exit animations mounted long
enough to finish.

Shared `--duration-*`, `--ease-*`, `--transition-*`, and `--scale-*` tokens affect multiple
components. Prefer component-local motion variables when the change should stay isolated to one
surface family.

When the operating system requests reduced motion, the foundation reduces shared duration tokens
to `0.01ms` and disables the shared pulse, fade, slide, and spin animations. Components with
specialized continuous motion, such as marquee and indeterminate progress, keep their own
`prefers-reduced-motion` fallback.

## Motion model

- `*-starting-*` values apply during entry and `*-ending-*` values apply during exit.
- Put motion variables on the part that visually animates, such as `Popover.Content`,
  `Dialog.Content`, `Menu.Content`, or a component-specific popup part.
- Reuse the same recipe across families by changing only the variable prefix, for example
  `--popover-content-*`, `--dialog-content-*`, `--menu-popup-*`, or `--tooltip-popup-*`.
- Use `present` only when JavaScript, not CSS, should control the mounted lifecycle.

## Ready Recipes

Apply these variables to `Popover.Content` through `className`, or swap the variable prefix for the
matching component family.

### Fade

```css
.fadeMotion {
  --popover-content-starting-opacity: 0;
  --popover-content-ending-opacity: 0;
  --popover-content-starting-scale: 1;
  --popover-content-ending-scale: 1;
  --popover-content-starting-translate-y: 0;
  --popover-content-ending-translate-y: 0;
}
```

### Slide Up

```css
.slideUpMotion {
  --popover-content-starting-opacity: 0;
  --popover-content-ending-opacity: 0;
```

## Usage Rules

- Put motion variables on the visual part that owns the animation.
- Keep entry and exit values symmetric unless asymmetric motion is intentional.
- Set scale to `1` for pure fade or slide motion.
- Prefer component-local overrides before changing shared duration or easing tokens globally.
- Ark presence keeps closed content mounted while CSS exit animations run. Use `present` only when
  JavaScript must control that lifecycle.
