# Components

moduix components are styled, composable Ark UI wrappers with a shared token language. Start from
the product need below, then use the sidebar or search for the complete catalog. Every component
page shows the public part tree, examples, Ark-aligned behavior, and local CSS variables.

## Before you compose

- Use [Quick start](/docs/quick-start) to choose package-managed or copy-owned installation.
- Read [Composition Patterns](/docs/composition-patterns) before hiding component parts in product
  wrappers.
- Use [Themes](/docs/themes) and [Tokens](/docs/tokens) for system-wide visual changes; keep
  component aliases for deliberate exceptions.
- Start from [Blocks](/docs/blocks) when you need a complete product screen rather than one
  primitive.
