## API Reference

`List` keeps native `ul`/`ol` semantics and adds design-system control over markers,
spacing, typography, and tone. `ListItem` is optional: use it for the exported item
slot, or render plain `li` elements directly.

## Anatomy

`List` renders a semantic list root with direct `li` children. The root owns marker
style, spacing, and typography tokens.

```text
List (ul | ol)
└─ ListItem
```

| Part       | Role                                                                   |
| ---------- | ---------------------------------------------------------------------- |
| `List`     | Semantic list root that controls marker, spacing, size, and tone.      |
| `ListItem` | Individual item wrapper rendered as native list item semantics (`li`). |

## Composition

Use `as` for list semantics, `marker` for marker style or visibility, and `size`,
`gap`, `tone`, plus `className` for visual customization.

The default marker follows the root element: `ul` uses `disc`, `ol` uses `decimal`.
Use `marker="none"` to keep list semantics without visible markers. Use
`marker="bullet"` when marker size, gap, radius, or vertical offset must be styled
independently from the text. In that mode, `ListItem` exposes the marker/content split,
while plain `li` elements still inherit root spacing and typography.

### Defaults

| Prop     | Default   | Values                                                 |
| -------- | --------- | ------------------------------------------------------ |
| `as`     | `ul`      | `ul`, `ol`                                             |
| `marker` | by `as`   | `disc`, `decimal`, `bullet`, `none`                    |
| `gap`    | `sm`      | `xs`, `sm`, `md`, `lg`, `xl`, `2xl`                    |
| `size`   | `md`      | `xs`, `sm`, `md`, `lg`, `xl`                           |
| `tone`   | `default` | `default`, `muted`, `subtle`, `primary`, `destructive` |

### Custom Composition

Use `marker="bullet"` when marker styling must be controlled independently from the text.

  <Preview.CSS>

        --list-marker-offset-y: 0.5em;
