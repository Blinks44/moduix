# React Hook Form

Use `register` for parts that render a native input. Use `Controller` when an Ark component exposes
controlled values and detail callbacks.

```bash
npm install react-hook-form
```

## Mapping

| Control                                         | React Hook Form wiring                                                    |
| ----------------------------------------------- | ------------------------------------------------------------------------- |
| `Field.Input`, `Field.Textarea`, `Field.Select` | Spread `register(name, rules)` onto the native control.                   |
| `Select`                                        | Use `Controller`; map a scalar string to Ark's string-array `value`.      |
| `Combobox`                                      | Use `Controller`; map `value` and keep filtering in `onInputValueChange`. |
| `Checkbox`                                      | Use `Controller`; map `details.checked === true` to a boolean.            |

Always provide `defaultValues` for controlled fields. React Hook Form uses them for initial state,
dirty checks, and `reset()`.

For shared or cross-field validation, add a schema resolver. The component mapping below stays the
same.

## Complete form

The form combines registered `Input` and `Textarea` controls with controlled `Select`, `Combobox`,
and `Checkbox`. `noValidate` lets React Hook Form own the error messages in this demo. Omit it when
you intentionally want the browser's native validation UI.

    {`
      import { createListCollection, useListCollection } from "@ark-ui/react/collection";
      import { useFilter } from "@ark-ui/react/locale";
      import {
        Button,
        Card,
        Checkbox,
        Combobox,
        Field,
        Portal,
        Select,
      } from "@moduix/react";
      import { Controller, useForm } from "react-hook-form";

      type FormValues = {
        name: string;
        team: string;
        reviewer: string;
        summary: string;
        notifications: boolean;
      };

      export function ProjectForm() {
        const { contains } = useFilter({ sensitivity: "base" });
        const { collection, filter } = useListCollection({
          initialItems: people,
          filter: contains,
        });
        const {
          control,
          handleSubmit,
          register,
          formState: { errors },
        } = useForm({
          defaultValues: {
            name: "",
            team: "",
            reviewer: "",
            summary: "",
            notifications: false,
          } as FormValues,
        });

        return (
          <form
            className="form"
            noValidate
            onSubmit={handleSubmit((values) => console.log(values))}
          >
            <Card>
              <Card.Header>
                <Card.Title>Create project</Card.Title>
                <Card.Description>
                  Share the details your team needs to get started.
                </Card.Description>
              </Card.Header>

              <Card.Body className="fields">
                <Field invalid={Boolean(errors.name)} required>
                  <Field.Label>
                    Project name
                    <Field.RequiredIndicator />
                  </Field.Label>
                  <Field.Input
                    {...register("name", {
                      required: "Enter a project name.",
                    })}
                  />
                  <Field.ErrorText>{errors.name?.message}</Field.ErrorText>
                </Field>

                <Controller
                  name="team"
                  control={control}
                  rules={{ required: "Choose a team." }}
                  render={({ field, fieldState }) => (
                    <Field invalid={fieldState.invalid} required>
                      <Field.Label>
                        Team
                        <Field.RequiredIndicator />
                      </Field.Label>
                      <Select
                        collection={teams}
                        name={field.name}
                        value={field.value ? [field.value] : []}
                        onValueChange={(details) =>
                          field.onChange(details.value[0] ?? "")
                        }
                        onInteractOutside={field.onBlur}
                      >
                        <Select.Control>
                          <Select.Trigger ref={field.ref}>
                            <Select.ValueText placeholder="Choose a team" />
                          </Select.Trigger>
                          <Select.Indicators>
                            <Select.Indicator />
                          </Select.Indicators>
                        </Select.Control>
                        <Portal>
                          <Select.Positioner>
                            <Select.Content>
                              {teams.items.map((item) => (
                                <Select.Item key={item.value} item={item}>
                                  <Select.ItemText>{item.label}</Select.ItemText>
                                  <Select.ItemIndicator />
                                </Select.Item>
                              ))}
                            </Select.Content>
                          </Select.Positioner>
                        </Portal>
                        <Select.HiddenSelect />
                      </Select>
                      <Field.ErrorText>
                        {fieldState.error?.message}
                      </Field.ErrorText>
                    </Field>
                  )}
                />

                <Controller
                  name="reviewer"
                  control={control}
                  rules={{ required: "Choose a reviewer." }}
                  render={({ field, fieldState }) => (
                    <Field invalid={fieldState.invalid} required>
                      <Field.Label>
                        Reviewer
                        <Field.RequiredIndicator />
                      </Field.Label>
                      <Combobox
                        collection={collection}
                        name={field.name}
                        value={field.value ? [field.value] : []}
                        onValueChange={(details) =>
                          field.onChange(details.value[0] ?? "")
                        }
                        onInputValueChange={(details) =>
                          filter(details.inputValue)
                        }
                      >
                        <Combobox.Control>
                          <Combobox.Input
                            ref={field.ref}
                            onBlur={field.onBlur}
                            placeholder="Search people"
                          />
                          <Combobox.ClearTrigger aria-label="Clear reviewer" />
                          <Combobox.Trigger aria-label="Open reviewers" />
                        </Combobox.Control>
                        <Portal>
                          <Combobox.Positioner>
                            <Combobox.Content>
                              <Combobox.Empty>No reviewers found.</Combobox.Empty>
                              <Combobox.List>
                                {collection.items.map((item) => (
                                  <Combobox.Item key={item.value} item={item}>
                                    <Combobox.ItemText>{item.label}</Combobox.ItemText>
                                    <Combobox.ItemIndicator />
                                  </Combobox.Item>
                                ))}
                              </Combobox.List>
                            </Combobox.Content>
                          </Combobox.Positioner>
                        </Portal>
                      </Combobox>
                      <Field.ErrorText>
                        {fieldState.error?.message}
                      </Field.ErrorText>
                    </Field>
                  )}
                />

                <Field>
                  <Field.Label>Summary</Field.Label>
                  <Field.Textarea
                    {...register("summary")}
                    placeholder="What are you planning to build?"
                    rows={3}
                  />
                </Field>

                <Controller
                  name="notifications"
                  control={control}
                  render={({ field }) => (
                    <Checkbox
                      name={field.name}
                      checked={field.value}
                      onCheckedChange={(details) =>
                        field.onChange(details.checked === true)
                      }
                    >
                      <Checkbox.Control>
                        <Checkbox.Indicator />
                      </Checkbox.Control>
                      <Checkbox.Label>Send status notifications</Checkbox.Label>
                      <Checkbox.HiddenInput
                        ref={field.ref}
                        onBlur={field.onBlur}
                      />
                    </Checkbox>
                  )}
                />
              </Card.Body>

              <Card.Footer>
                <Button className="submit" type="submit">
                  Create project
                </Button>
              </Card.Footer>
            </Card>
          </form>
        );
      }
    `}

## Important details

- Do not spread a `Controller` field object onto `Select` or `Combobox`; their callback and value
  shapes are not native input events.
- For single selection, pass `field.value ? [field.value] : []` and read
  `details.value[0] ?? ""`.
- Call `field.onBlur` from `Select.onInteractOutside` or `Combobox.Input.onBlur`.
- Forward `field.ref` to `Select.Trigger` or `Combobox.Input`. This lets React Hook Form focus the
  first invalid field.
- Keep `Select.HiddenSelect` mounted for native semantics, autofill, and reset synchronization.
- Keep `Checkbox.HiddenInput` mounted and map its checked state to a boolean field value.
