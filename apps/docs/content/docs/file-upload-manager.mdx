# File upload manager

Add a polished attachment surface with drag-and-drop, a native file picker, file limits, and clear
feedback when a file is not accepted. The block manages the local list only: connect your own upload
mutation where the product needs to send files to a server.

## Install

<ShadcnInstall packageName="file-upload-manager" itemLabel="block" />

The command copies the component and its CSS module into your configured components directory and
installs `@moduix/react` when needed.

## Connect an upload

`FileUpload` keeps the selected files in its local state. Add `onFileChange` to the root when your
route needs to pass `details.acceptedFiles` to a form action or upload mutation. Keep the progress
and completion state next to that real request rather than simulating it in the block.
