`node_modules`, install it from the moduix GitHub registry:

## Anatomy

`Breadcrumbs` is a small set of semantic building blocks.

```text
Breadcrumbs
└─ BreadcrumbsList
   ├─ BreadcrumbsItem
   │  └─ BreadcrumbsLink
   ├─ BreadcrumbsSeparator
   └─ BreadcrumbsItem
      └─ BreadcrumbsPage
```

| Part                   | Role                                     |
| ---------------------- | ---------------------------------------- |
| `Breadcrumbs`          | Root navigation region.                  |
| `BreadcrumbsList`      | Ordered list of breadcrumb segments.     |
| `BreadcrumbsItem`      | List item wrapper for a segment.         |
| `BreadcrumbsLink`      | Styled link for navigable segments.      |
| `BreadcrumbsPage`      | Current page text with `aria-current`.   |
| `BreadcrumbsSeparator` | Visual delimiter between visible items.  |
| `BreadcrumbsEllipsis`  | Minimal collapsed marker for menu usage. |

## Composition

Use `BreadcrumbsLink` for navigable segments and `BreadcrumbsPage` for the current page.

Use plain `href` when a normal anchor is enough. Use `render={<Link />}` when the app
standardizes on a router link component. Keep collapsed paths explicit by composing `Breadcrumbs`
with `Menu`; the breadcrumb parts themselves stay stateless and semantic.

### Custom Separator

Replace the default slash with your own icon or symbol.

<Preview>
  <BreadcrumbsCustomSeparatorExample />

### Collapsed Path

Compose `Breadcrumbs` with `Menu` when you need a collapsed middle segment.

<Preview>
  <BreadcrumbsCollapsedExample />

  <Preview.Code>
    {`
      import {
        Breadcrumbs,
        BreadcrumbsEllipsis,
        BreadcrumbsItem,
        BreadcrumbsLink,
        BreadcrumbsList,
        BreadcrumbsPage,
        BreadcrumbsSeparator,
        Menu,
        MenuContent,
        MenuLinkItem,
        MenuTrigger,
      } from "moduix";

      export function BreadcrumbsCollapsedDemo() {
        return (
          <Breadcrumbs>
            <BreadcrumbsList>
              <BreadcrumbsItem>
                <BreadcrumbsLink href="/">Home</BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <Menu>
                  <MenuTrigger
                    aria-label="Show hidden path items"
                    style={{
                      minWidth: "auto",
                      minHeight: "auto",
                      padding: 0,
                      border: "none",
                      background: "transparent",
                      color: "inherit",
                    }}
                  >
                    <BreadcrumbsEllipsis />
                  </MenuTrigger>
                  <MenuContent
                    align="start"
                    style={{
                      "--menu-popup-min-width": "8rem",
                      "--menu-item-font-size": "var(--text-xs)",
                      "--menu-item-padding-x-start": "0.625rem",
                      "--menu-item-padding-x-end": "0.625rem",
                      "--menu-item-padding-y": "0.25rem",
                    }}
                  >
                    <MenuLinkItem closeOnClick href="/engineering">Engineering</MenuLinkItem>
                    <MenuLinkItem closeOnClick href="/engineering/backend">Backend</MenuLinkItem>
                    <MenuLinkItem closeOnClick href="/engineering/backend/golang">Golang</MenuLinkItem>
                  </MenuContent>
                </Menu>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsLink href="/vacancies">Vacancies</BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsPage>Go Developer</BreadcrumbsPage>
              </BreadcrumbsItem>
            </BreadcrumbsList>
          </Breadcrumbs>
        );
      }
    `}

  </Preview.Code>
</Preview>
