## Anatomy

`Breadcrumbs` uses semantic `nav` + `ol` for path navigation.
Long paths collapse middle items into an ellipsis trigger with a compact Base UI menu.

```text
Breadcrumbs
└─ list
   ├─ item link
   ├─ separator
   ├─ collapsed menu trigger (optional)
   ├─ separator
   └─ current page item
```

| Part                     | Role                                                              |
| ------------------------ | ----------------------------------------------------------------- |
| `Breadcrumbs`            | Root navigation region with aria label and path layout.           |
| `item`                   | Breadcrumb segment, rendered as link or current-page text.        |
| `separator`              | Visual delimiter between visible segments.                        |
| `collapsed menu trigger` | Ellipsis control shown when `maxItems` collapses middle segments. |
| `collapsed menu popup`   | Compact list of hidden segments built with Base UI `Menu`.        |

Visible items stay on one line, and long labels are clamped.

## Composition

Use `items` to describe the path. Each item can be plain text, a regular link (`href`),
or an action item (`onClick`).

For framework routers, use `render` and return your own link while preserving provided
`className`, `children`, and navigation props such as `href`, `target`, `rel`, and `onClick`.
That same contract is used both in the visible breadcrumb row and in the collapsed menu, so
router links behave consistently when items move under the ellipsis.

`maxItems` controls how many middle segments stay visible. The first segment and current
page always stay visible; hidden middle segments move to the collapsed menu.

For collapsed items, style internal parts via `classNames`
(`ellipsisTrigger`, `popup`, `popupItem`, `popupLinkItem`, `portal`, `positioner`)
and pass behavior props to the same slots via `slotProps`.

| Prop                   | Default                  | Values         |
| ---------------------- | ------------------------ | -------------- |
| `separator`            | `/`                      | Any React node |
| `maxItems`             | `3`                      | Number >= `0`  |
| `ellipsisLabel`        | `...`                    | Any React node |
| `hiddenItemsMenuLabel` | `Show hidden path items` | String         |

## Examples

### Collapsed Path

Collapse middle segments into an ellipsis menu. Here `maxItems={2}` keeps two middle
segments visible next to the current page.

<Preview>
  <BreadcrumbsCollapsedExample />

    {`
      import { Breadcrumbs } from "moduix";

        return (
          <Breadcrumbs
            maxItems={2}
            items={[
              { label: "Home", href: "/" },
              { label: "Engineering", href: "/engineering" },
              { label: "Backend", href: "/engineering/backend" },
              { label: "Golang", href: "/engineering/backend/golang" },
              { label: "Vacancies", href: "/vacancies" },
              { label: "Go Developer" },

### Framework Links

Use `render` to integrate external routing links while keeping component styles. The same
pattern works for Next.js, TanStack Router, Inertia, and other router link components.

<Preview>
  <BreadcrumbsRenderExample />

  <Preview.Code>
    {`
      import { Breadcrumbs } from "moduix";
      import Link from "next/link";

      export function BreadcrumbsWithNextLinkDemo() {
        return (
          <Breadcrumbs
            maxItems={1}
            items={[
              {
                label: "Home",
                href: "/",
                render: ({ href, className, children, ...props }) => (
                  <Link href={href ?? "/"} className={className} {...props}>
                    {children}
                  </Link>
                ),
              },
              {
                label: "Engineering",
                href: "/engineering",
                render: ({ href, className, children, ...props }) => (
                  <Link href={href ?? "/engineering"} className={className} {...props}>
                    {children}
                  </Link>
                ),
              },
              {
                label: "Backend",
                href: "/engineering/backend",
                render: ({ href, className, children, ...props }) => (
                  <Link href={href ?? "/engineering/backend"} className={className} {...props}>
                    {children}
                  </Link>
                ),
              },
              { label: "Vacancies", href: "/vacancies" },
              { label: "Go Developer" },
            ]}
          />
        );
      }
    `}

  </Preview.Code>
</Preview>

### Action Items

Use `onClick` for intermediate path items that trigger navigation or another action without
rendering a plain anchor. The same handler is preserved when the item moves into the collapsed menu.

<Preview>
  <BreadcrumbsActionExample />

  <Preview.Code>
    {`
      import { Breadcrumbs } from "moduix";

      export function BreadcrumbsActionDemo() {
        return (
          <Breadcrumbs
            maxItems={1}
            items={[
              { label: "Home", href: "/" },
              { label: "Projects", onClick: () => console.log("Projects") },
              { label: "Backend", onClick: () => console.log("Backend") },
              { label: "Go Developer" },
            ]}
          />
        );
      }
    `}

  </Preview.Code>
</Preview>

### Collapsed Menu Slots

Use `classNames` and `slotProps` to customize collapsed trigger and popup internals.

<Preview>
  <BreadcrumbsSlotsExample />

  <Preview.Code>
    {`
      import { Breadcrumbs } from "moduix";

      export function BreadcrumbsSlotsDemo() {
        return (
          <Breadcrumbs
            maxItems={1}
            items={[
              { label: "Home", href: "/" },
              { label: "Engineering", href: "/engineering" },
              { label: "Backend", href: "/engineering/backend" },
              { label: "Vacancies", href: "/vacancies" },
              { label: "Go Developer" },
            ]}
            classNames={{
              ellipsisTrigger: styles.ellipsisTrigger,
              popup: styles.popup,
              popupItem: styles.popupItem,
            }}
            slotProps={{
              positioner: { sideOffset: 6 },
              popup: { "aria-label": "Hidden breadcrumb links" },
            }}
          />
        );
      }
    `}

  </Preview.Code>
</Preview>
