## Anatomy

`Breadcrumbs` is a small set of semantic building blocks.

```text
Breadcrumbs
└─ BreadcrumbsList
   ├─ BreadcrumbsItem
   │  └─ BreadcrumbsLink
   ├─ BreadcrumbsSeparator
   └─ BreadcrumbsItem
      └─ BreadcrumbsPage
```

| Part                   | Role                                     |
| ---------------------- | ---------------------------------------- |
| `Breadcrumbs`          | Root navigation region.                  |
| `BreadcrumbsList`      | Ordered list of breadcrumb segments.     |
| `BreadcrumbsItem`      | List item wrapper for a segment.         |
| `BreadcrumbsLink`      | Styled link for navigable segments.      |
| `BreadcrumbsPage`      | Current page text with `aria-current`.   |
| `BreadcrumbsSeparator` | Visual delimiter between visible items.  |
| `BreadcrumbsEllipsis`  | Minimal collapsed marker for menu usage. |

## Composition

Use `BreadcrumbsLink` for regular links and `BreadcrumbsPage` for the current page.

For framework routers, pass `render` to `BreadcrumbsLink` and return your own link while
forwarding the received props to the underlying DOM element.

Collapsed paths are composed explicitly with `Menu`. `Breadcrumbs` no longer owns collapsed state
or hidden-items logic.

### Collapsed Path

Compose `Breadcrumbs` with `Menu` when you need a collapsed middle segment.

<Preview>
  <BreadcrumbsCollapsedExample />

  <Preview.Code>
    {`
      import {
        Breadcrumbs,
        BreadcrumbsEllipsis,
        BreadcrumbsItem,
        BreadcrumbsLink,
        BreadcrumbsList,
        BreadcrumbsPage,
        BreadcrumbsSeparator,
        Menu,
        MenuContent,
        MenuLinkItem,
        MenuTrigger,
      } from "moduix";

      export function BreadcrumbsCollapsedDemo() {
        return (
          <Breadcrumbs>
            <BreadcrumbsList>
              <BreadcrumbsItem>
                <BreadcrumbsLink href="/">Home</BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <Menu>
                  <MenuTrigger aria-label="Show hidden path items">
                    <BreadcrumbsEllipsis />
                  </MenuTrigger>
                  <MenuContent
                    align="start"
                    withArrow={false}
                    style={{
                      "--menu-item-font-size": "var(--text-xs)",
                      "--menu-item-padding-x-end": "0.75rem",
                      "--menu-item-padding-x-start": "0.75rem",
                      "--menu-item-padding-y": "0.375rem",
                    }}
                  >
                    <MenuLinkItem closeOnClick href="/engineering">Engineering</MenuLinkItem>
                    <MenuLinkItem closeOnClick href="/engineering/backend">Backend</MenuLinkItem>
                    <MenuLinkItem closeOnClick href="/engineering/backend/golang">Golang</MenuLinkItem>
                  </MenuContent>
                </Menu>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsLink href="/vacancies">Vacancies</BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsPage>Go Developer</BreadcrumbsPage>
              </BreadcrumbsItem>
            </BreadcrumbsList>
          </Breadcrumbs>
        );
      }
    `}

  </Preview.Code>
</Preview>

### Framework Links

Use `render` to integrate router link components while keeping breadcrumb styles.

<Preview>
  <BreadcrumbsRenderExample />

  <Preview.Code>
    {`
      import Link from "next/link";
      import {
        Breadcrumbs,
        BreadcrumbsItem,
        BreadcrumbsLink,
        BreadcrumbsList,
        BreadcrumbsPage,
        BreadcrumbsSeparator,
      } from "moduix";

      export function BreadcrumbsWithNextLinkDemo() {
        return (
          <Breadcrumbs>
            <BreadcrumbsList>
              <BreadcrumbsItem>
                <BreadcrumbsLink
                  href="/"
                  render={({ href, className, children, ...props }) => (
                    <Link href={href ?? "/"} className={className} {...props}>
                      {children}
                    </Link>
                  )}
                >
                  Home
                </BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsLink
                  href="/vacancies"
                  render={({ href, className, children, ...props }) => (
                    <Link href={href ?? "/vacancies"} className={className} {...props}>
                      {children}
                    </Link>
                  )}
                >
                  Vacancies
                </BreadcrumbsLink>
              </BreadcrumbsItem>
              <BreadcrumbsSeparator />
              <BreadcrumbsItem>
                <BreadcrumbsPage>Go Developer</BreadcrumbsPage>
              </BreadcrumbsItem>
            </BreadcrumbsList>
          </Breadcrumbs>
        );
      }
    `}

  </Preview.Code>
</Preview>
