## Anatomy

`Menubar` keeps the high-level path short. Most consumers only need top-level menus, visible items,
and submenu parts.

```text
Menubar
â””â”€ MenubarMenu
   â”śâ”€ MenubarTrigger
   â””â”€ MenubarContent
      â”śâ”€ MenubarItem / MenubarLinkItem
      â”śâ”€ MenubarCheckboxItem / MenubarRadioItem
      â”śâ”€ MenubarGroup
      â”śâ”€ MenubarSeparator
      â””â”€ MenubarSubmenu
         â”śâ”€ MenubarSubmenuTrigger
         â””â”€ MenubarSubmenuContent
```

| Part                                                                                           | Role                                                                                                             |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `Menubar`                                                                                      | Root state machine for orientation, roving focus, and top-level menu behavior.                                   |
| `MenubarMenu`                                                                                  | One top-level menu scope that links a trigger and popup content.                                                 |
| `MenubarTrigger`                                                                               | Interactive trigger inside the menubar row or column.                                                            |
| `MenubarContent`                                                                               | Default popup wrapper with built-in `MenubarPortal`, `MenubarPositioner`, `MenubarPopup`, and `MenubarViewport`. |
| `MenubarItem` / `MenubarLinkItem`                                                              | Action and navigation rows.                                                                                      |
| `MenubarCheckboxItem` / `MenubarRadioItem`                                                     | Stateful command rows with indicator slots.                                                                      |
| `MenubarItemText*` / `MenubarItemShortcut`                                                     | Optional layout helpers for icons, labels, and shortcut text.                                                    |
| `MenubarSeparator` / `MenubarGroup*`                                                           | Visual grouping parts for related commands.                                                                      |
| `MenubarSubmenu`                                                                               | Nested command group with its own trigger and popup content.                                                     |
| `MenubarPortal` / `MenubarPositioner` / `MenubarPopup` / `MenubarViewport` / `MenubarBackdrop` | Low-level popup parts for advanced composition.                                                                  |

## Composition

Each `MenubarMenu` should contain one `MenubarTrigger` and one popup branch. Start with
`MenubarContent`. It keeps the markup close to Base UI while covering the common path:

- portal mounting
- popup positioning
- scrollable viewport
- optional built-in arrow via `showArrow`

Use `MenubarSubmenuContent` for nested menus. It keeps the same API as `MenubarContent` and adds
side-aware submenu offsets by default.

When you need a custom portal container, backdrop, popup layout, or manual arrow placement, switch
to the exported low-level parts and compose the popup yourself.

### Small Sugar

| Surface                                    | Prop          | Default                                                     | Why it exists                                                               |
| ------------------------------------------ | ------------- | ----------------------------------------------------------- | --------------------------------------------------------------------------- |
| `MenubarContent`                           | `showArrow`   | `false`                                                     | Turns on the built-in popup arrow.                                          |
| `MenubarContent`                           | `sideOffset`  | `6`                                                         | Gives the root popup a usable default gap.                                  |
| `MenubarSubmenuContent`                    | `showArrow`   | `false`                                                     | Same arrow toggle for submenu popups.                                       |
| `MenubarSubmenuContent`                    | `sideOffset`  | by `side` (`4` for `top`/`bottom`, `-4` for `left`/`right`) | Keeps submenu spacing aligned by default.                                   |
| `MenubarSubmenuContent`                    | `alignOffset` | by `side` (`4` for `top`/`bottom`, `-4` for `left`/`right`) | Keeps submenu alignment aligned by default.                                 |
| `MenubarCheckboxItem` / `MenubarRadioItem` | `indicator`   | `start`                                                     | Lets checked-state indicators move to the end without changing composition. |

## Examples

### Groups And Controls

Use groups, radio items, and checkbox items when menubar commands change application state.

<Preview>
  <GroupsAndControlsMenubarExample />

  <Preview.Code>
    {`
      import {
        Menubar,
        MenubarCheckboxItem,
        MenubarCheckboxItemIndicator,
        MenubarContent,
        MenubarGroup,
        MenubarGroupLabel,
        MenubarItemText,
        MenubarMenu,
        MenubarRadioGroup,
        MenubarRadioItem,
        MenubarRadioItemIndicator,
        MenubarSeparator,
        MenubarTrigger,
      } from "moduix";
      import { useState } from "react";

      export function GroupsAndControlsMenubarDemo() {
        const [sortBy, setSortBy] = useState("name");
        const [showSidebar, setShowSidebar] = useState(true);
        const [showPanel, setShowPanel] = useState(false);

        return (
          <Menubar>
            <MenubarMenu>
              <MenubarTrigger>View</MenubarTrigger>
              <MenubarContent>
                <MenubarGroup>
                  <MenubarGroupLabel>Sort</MenubarGroupLabel>
                  <MenubarRadioGroup value={sortBy} onValueChange={setSortBy}>
                    <MenubarRadioItem value="name">
                      <MenubarRadioItemIndicator />
                      <MenubarItemText>Name</MenubarItemText>
                    </MenubarRadioItem>
                    <MenubarRadioItem value="date">
                      <MenubarRadioItemIndicator />
                      <MenubarItemText>Date Modified</MenubarItemText>
                    </MenubarRadioItem>
                    <MenubarRadioItem value="type">
                      <MenubarRadioItemIndicator />
                      <MenubarItemText>Type</MenubarItemText>
                    </MenubarRadioItem>
                  </MenubarRadioGroup>
                </MenubarGroup>

                <MenubarSeparator />

                <MenubarGroup>
                  <MenubarGroupLabel>Panels</MenubarGroupLabel>
                  <MenubarCheckboxItem checked={showSidebar} onCheckedChange={setShowSidebar}>
                    <MenubarCheckboxItemIndicator />
                    <MenubarItemText>Sidebar</MenubarItemText>
                  </MenubarCheckboxItem>
                  <MenubarCheckboxItem checked={showPanel} onCheckedChange={setShowPanel}>
                    <MenubarCheckboxItemIndicator />
                    <MenubarItemText>Preview</MenubarItemText>
                  </MenubarCheckboxItem>
                </MenubarGroup>
              </MenubarContent>
            </MenubarMenu>
          </Menubar>
        );
      }
    `}

  </Preview.Code>
</Preview>

## Custom Composition

Use the low-level popup parts when the default wrapper is not enough. This is the path for custom
portal control, backdrops, arrow placement, or popup-specific styling.

        MenubarPortal,
        MenubarPositioner,
        MenubarSeparator,
        MenubarTrigger,
        MenubarViewport,
      } from "moduix";
      import styles from "./menubar.module.css";

      export function CustomCompositionMenubarDemo() {
        return (
