## API Reference

`Kbd` is a standalone moduix component and does not wrap a Base UI primitive.
Use it to display keyboard shortcuts, single keys, and small command hints.

## Anatomy

Use `Kbd` for a single key or command label. Use `KbdGroup` when multiple keys must stay aligned
as one shortcut.

```text
KbdGroup
└─ Kbd
└─ Kbd
```

```tsx
<KbdGroup>
  <Kbd>Cmd</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

| Part       | Role                                                       |
| ---------- | ---------------------------------------------------------- |
| `Kbd`      | Root keycap slot. Applies size, variant, border, and tone. |
| `KbdGroup` | Inline wrapper for keyboard shortcut combinations.         |

`Kbd` does not use service slots such as `portal`, `positioner`, `backdrop`, or `viewport`.

## Composition

Pass `className` to `Kbd` or `KbdGroup` for targeted overrides. `Kbd` accepts standard native
`kbd` props, so you can add `title`, `aria-*`, and testing attributes without wrappers. `KbdGroup`
accepts standard `span` props.

### Defaults

| Part       | Prop      | Default   | Values                        |
| ---------- | --------- | --------- | ----------------------------- |
| `Kbd`      | `variant` | `default` | `default`, `outline`, `ghost` |
| `Kbd`      | `size`    | `md`      | `sm`, `md`, `lg`              |
| `KbdGroup` | `size`    | `md`      | `sm`, `md`, `lg`              |

## Examples

### Variants

Use `default` for primary keycap hints, `outline` for quieter surfaces, and `ghost` for dense menus.

<Preview>
  <KbdVariantsExample />

  <Preview.Code>
    {`
      import { Kbd } from "moduix";

      export function KbdVariantsDemo() {
        return (
          <>
            <Kbd variant="default">default</Kbd>
            <Kbd variant="outline">outline</Kbd>
            <Kbd variant="ghost">ghost</Kbd>
          </>
        );
      }
    `}

  </Preview.Code>
</Preview>

### Sizes

Switch `size` between `sm`, `md`, and `lg` to match dense lists, toolbars, and command panels.

<Preview>
  <KbdSizesExample />

  <Preview.Code>
    {`
      import { Kbd } from "moduix";

      export function KbdSizesDemo() {
        return (
          <>
            <Kbd size="sm">Esc</Kbd>
            <Kbd size="md">Cmd K</Kbd>
            <Kbd size="lg">Enter</Kbd>
          </>
        );
      }
    `}

  </Preview.Code>
</Preview>

### Shortcut Group

Use `KbdGroup` to keep multi-key shortcuts together.

<Preview>
  <KbdGroupExample />

  <Preview.Code>

        return (
          <KbdGroup>
            <Kbd>Cmd</Kbd>
            <Kbd>K</Kbd>
          </KbdGroup>

  </Preview.Code>
</Preview>

Use `className` and CSS variables for product-specific shortcut styling.
