## API Reference

`Kbd` is a standalone moduix component and does not wrap a Base UI primitive.
Because Base UI does not ship a dedicated keyboard-key primitive, moduix keeps this component as a
thin styled wrapper around the native `<kbd>` element. Use `Kbd` for a single key and `KbdGroup`
when several keys need to read as one shortcut.

## Anatomy

Use `Kbd` for a single key label. Use `KbdGroup` to keep multi-key shortcuts aligned.

```text
KbdGroup
└─ Kbd
└─ Kbd
```

```tsx
<KbdGroup>
  <Kbd>Cmd</Kbd>+<Kbd>K</Kbd>
</KbdGroup>
```

| Part       | Role                                                 |
| ---------- | ---------------------------------------------------- |
| `Kbd`      | Root keycap slot. Applies the shared keycap styling. |
| `KbdGroup` | Inline wrapper for keyboard shortcut combinations.   |

## Composition

Pass `className` to `Kbd` or `KbdGroup` for targeted overrides. `Kbd` accepts native `kbd` props,
and `KbdGroup` accepts native `span` props.

There is no extra variant or size API. For denser layouts or product-specific visuals, use
`className` and the public `--kbd-*` CSS variables.

### Compact Density

Use local CSS variables for one-off density or tone changes instead of extra component props.

<Preview>
  <KbdDenseExample />

### Custom Composition

Use `className` and CSS variables when the default keycap needs to match product-specific styling.

<Preview>
