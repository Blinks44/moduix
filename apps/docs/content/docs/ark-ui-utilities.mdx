# Ark UI Utilities

Ark UI also ships utility components and providers that do not map directly to styled moduix
components. Use them directly from Ark when you need the behavior and the relevant moduix component
page does not document a wrapper.

```tsx
import { ClientOnly, Format, Presence } from '@ark-ui/react';
```

Some utilities have dedicated subpath imports, such as `@ark-ui/react/environment` and
`@ark-ui/react/locale`. Check the linked Ark page for the exact import and API.

## Overview

| Ark utility                                                                    | Use it for                                                                                                |
| ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------- |
| [Client Only](https://ark-ui.com/docs/utilities/client-only)                   | Rendering browser-only content after the client has mounted, with optional fallback content.              |
| [Download Trigger](https://ark-ui.com/docs/utilities/download-trigger)         | Triggering downloads for URLs, blobs, files, strings, or promise-returned downloadable data.              |
| [Environment](https://ark-ui.com/docs/utilities/environment)                   | Providing the correct document/root node for iframes, Shadow DOM, Electron, or other custom environments. |
| [Focus Trap](https://ark-ui.com/docs/utilities/focus-trap)                     | Keeping keyboard focus inside a container for custom modal or attention-demanding interfaces.             |
| [Format Byte](https://ark-ui.com/docs/utilities/format-byte)                   | Formatting byte values with locale, unit, unit display, and decimal or binary unit systems.               |
| [Format Time](https://ark-ui.com/docs/utilities/format-time)                   | Formatting `Date` values or `HH:mm[:ss]` strings with locale-aware time options.                          |
| [Format Relative Time](https://ark-ui.com/docs/utilities/format-relative-time) | Rendering relative time through `Intl.RelativeTimeFormat`, including short display styles.                |
| [Frame](https://ark-ui.com/docs/utilities/frame)                               | Rendering React content inside an iframe, with head injection and mount lifecycle callbacks.              |
| [Highlight](https://ark-ui.com/docs/utilities/highlight)                       | Marking matched text ranges for search and filtering interfaces.                                          |
| [JSON Tree View](https://ark-ui.com/docs/utilities/json-tree-view)             | Inspecting JSON-like data in an expandable tree with selection, expansion, and custom value rendering.    |
| [Locale](https://ark-ui.com/docs/utilities/locale)                             | Setting app locale and direction for Ark formatting and locale-aware behavior.                            |
| [Presence](https://ark-ui.com/docs/utilities/presence)                         | Controlling lazy mount, exit animation completion, and unmount behavior around visible state.             |
| [Swap](https://ark-ui.com/docs/utilities/swap)                                 | Animating between two visual states, usually for icons or compact indicators.                             |

## Contract Boundary

These utilities come from Ark UI, not from the moduix styled component layer. That means Ark owns
their props, lifecycle, accessibility details, and release cadence. moduix component pages call out
the places where a utility is already part of a component contract, such as presence behavior on
overlays or collection data for list controls.
