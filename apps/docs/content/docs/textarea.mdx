`Textarea` stays intentionally small: it renders one styled native `<textarea>`, works directly with
`Field`, and keeps the native textarea contract intact.

## Anatomy

`Textarea` is a single native `<textarea>` root that composes with `Field`.

```text
Field (optional wrapper)
└─ Textarea
```

| Part       | Role                                                       |
| ---------- | ---------------------------------------------------------- |
| `Textarea` | Native textarea element with moduix styles and data attrs. |

## Composition

`Textarea` renders through Base UI `Field.Control`, so it works with `FieldLabel`,
`FieldDescription`, `FieldError`, native validation, and standard textarea props such as `value`,
`defaultValue`, `onValueChange`, `rows`, `maxLength`, `name`, and `readOnly`.

Use these paths:

- default `Textarea` path for regular multiline input
- `Textarea` inside `Field` when labels, descriptions, and validation should stay declarative
- `autoResize` when the field should grow with content in browsers that support
  `field-sizing: content`

`Textarea` does not add slot props, render helpers, or a parallel customization API. Style the root
directly with `className` or the public `--textarea-*` CSS variables.

### Defaults

| Prop            | Default | Notes                                                                |
| --------------- | ------- | -------------------------------------------------------------------- |
| `autoResize`    | `false` | Keeps the native resize behavior unless CSS auto-resize is opted in. |
| `onValueChange` | -       | Called with the next textarea value on change.                       |
| `className`     | -       | Applied directly to the visible textarea root.                       |

### Readonly Editing

For multiline inline editing, keep the same `Textarea` mounted and switch only `readOnly` plus the
action buttons. That avoids the text-to-textarea layout jump and lets you focus the field
immediately when edit mode starts.

<Preview>
  <TextareaReadOnlyEditingExample />

  <Preview.Code>
    {`
      import {
        Button,
        CheckIcon,
        CloseIcon,
        Field,
        FieldDescription,
        FieldLabel,
        PencilIcon,
        Textarea,
      } from "moduix";
      import { useEffect, useRef, useState } from "react";

      export function ReadonlyEditingTextareaDemo() {
        const [editing, setEditing] = useState(false);
        const [value, setValue] = useState(
          "Build the docs examples first, then harden the public API around real usage.",
        );
        const [draft, setDraft] = useState(value);
        const textareaRef = useRef(null as HTMLTextAreaElement | null);

        useEffect(() => {
          if (editing) {
            textareaRef.current?.focus();
          }
        }, [editing]);

        const handleEdit = () => {
          setDraft(value);
          setEditing(true);
        };

        const handleCancel = () => {
          setDraft(value);
          setEditing(false);
        };

        const handleSave = () => {
          setValue(draft);
          setEditing(false);
        };

        return (
          <Field className={styles.field}>
            <FieldLabel>Team note</FieldLabel>
            <FieldDescription>
              The textarea stays mounted and only switches between read-only and editable modes.
            </FieldDescription>
            <Textarea
              ref={textareaRef}
              autoResize
              readOnly={!editing}
              rows={3}
              value={editing ? draft : value}
              onValueChange={setDraft}
            />
            <div className={styles.actions}>
              {editing ? (
                <>
                  <Button aria-label="Cancel editing" size="icon-md" variant="ghost" onClick={handleCancel}>
                    <CloseIcon />
                  </Button>
                  <Button aria-label="Save changes" size="icon-md" onClick={handleSave}>
                    <CheckIcon />
                  </Button>
                </>
              ) : (
                <Button aria-label="Edit team note" size="icon-md" variant="ghost" onClick={handleEdit}>
                  <PencilIcon />
                </Button>
              )}
            </div>
          </Field>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .field {
        width: min(28rem, calc(100vw - var(--spacing-8)));
      }

      .actions {
        display: flex;
        justify-content: flex-end;
        gap: var(--spacing-2);
      }
    `}

  </Preview.CSS>
</Preview>
