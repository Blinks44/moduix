## Anatomy

`Textarea` is a single native `<textarea>` root that composes with `Field`.

```text
Field (optional wrapper)
└─ Textarea
```

| Part       | Role                                                       |
| ---------- | ---------------------------------------------------------- |
| `Textarea` | Native textarea element with moduix styles and data attrs. |

## Composition

`Textarea` renders through Base UI `Field.Control`, so it works with `FieldLabel`,
`FieldDescription`, `FieldError`, native validation, and standard textarea props such as `value`,
`defaultValue`, `onChange`, `rows`, `maxLength`, and `name`.

Use `size` for visual sizing, `resize` for native resize behavior, and `autoResize` to grow with
content while disabling the manual resize handle:

```tsx
<Textarea size="md" resize="vertical" autoResize />
```

For styling, pass `className` directly to `Textarea`.

### Defaults

| Prop         | Default    | Values                                   |
| ------------ | ---------- | ---------------------------------------- |
| `size`       | `md`       | `xs`, `sm`, `md`, `lg`, `xl`             |
| `resize`     | `vertical` | `none`, `both`, `horizontal`, `vertical` |
| `autoResize` | `false`    | `true`, `false`                          |

## Examples

### Controlled

Use `value` and `onChange` when content should stay in React state.

<Preview>
  <ControlledTextareaExample />

  <Preview.Code>
    {`
      import { Field, FieldLabel, Textarea } from "moduix";
      import { useState } from "react";

      function ControlledTextarea() {
        const [value, setValue] = useState("");

        return (
          <Field className={styles.field}>
            <FieldLabel>Feedback</FieldLabel>
            <Textarea
              value={value}
              onChange={(event) => setValue(event.currentTarget.value)}
              placeholder="Type to control value"
            />
          </Field>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .field {
        width: min(28rem, calc(100vw - var(--spacing-8)));
      }
    `}

  </Preview.CSS>
</Preview>
