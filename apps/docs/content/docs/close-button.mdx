## Anatomy

`CloseButton` is a single icon-button slot. By default it renders `CloseIcon`; pass `children` to replace it with other icon-sized content.

```text
CloseButton
└─ icon (default) | custom children
```

| Part          | Role                                                                   |
| ------------- | ---------------------------------------------------------------------- |
| `CloseButton` | Interactive close control built on Button semantics and accessibility. |

## Composition

`CloseButton` is a thin wrapper over Base UI `Button`.

- Use `children` to replace the default icon with another icon-sized child.
- Use `className` to style the root.
- Use any Base UI `Button` prop directly when you need native composition behavior such as `render`,
  `nativeButton`, `disabled`, or `focusableWhenDisabled`.

When `children` are omitted, the default accessible name is `Close`. When you render a custom
icon-like child, set `aria-label` to describe the dismissed surface.

| Prop         | Default                             | Values                                          |
| ------------ | ----------------------------------- | ----------------------------------------------- |
| `type`       | `button`                            | Any valid button `type` value                   |
| `aria-label` | `Close` when `children` are omitted | Any descriptive label for the dismissed surface |
| `children`   | `CloseIcon`                         | Any icon-sized React node                       |

Use `className` to style the root. `classNames` is not needed because there are no extra service slots.

## Examples

### Custom Children

Replace the default icon via `children`.

<Preview>
  <CloseButtonCustomChildrenExample />

  <Preview.Code>
    {`
      import { CloseButton } from "moduix";

      function CircleXIcon() {
        return (
          <svg
            xmlns="http://www.w3.org/2000/svg"
            viewBox="0 0 24 24"
            fill="none"
            aria-hidden="true"
            focusable="false"
            stroke="currentColor"
            strokeWidth="3"
            strokeLinecap="round"
            strokeLinejoin="round"
          >
            <circle cx="12" cy="12" r="10" />
            <path d="m15 9-6 6" />
            <path d="m9 9 6 6" />
          </svg>
        );
      }

      export function CloseButtonCustomChildrenDemo() {
        return (
          <CloseButton className={styles.customChildrenButton} aria-label="Close panel">
            <CircleXIcon />
          </CloseButton>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .customChildrenButton {
        --close-button-size: 2.25rem;
        --close-button-icon-size: 1.125rem;
      }
    `}

  </Preview.CSS>
</Preview>
