## API Reference

`Spinner` is a moduix loading primitive and does not wrap a Base UI primitive.
Use it for pending controls, small empty states, and inline loading feedback. It renders status
semantics by default and can be marked decorative when nearby text already announces loading.

## Anatomy

`Spinner` has one public root and an internal indicator slot. The indicator contains either the
selected built-in glyph or the custom `icon` node.

```text
Spinner
└─ indicator
```

```tsx
<Spinner variant="ring" size="md" />
```

| Part                | Role                                                                 |
| ------------------- | -------------------------------------------------------------------- |
| `Spinner`           | Root status element. Use `variant`, `size`, `label`, and `icon`.     |
| `spinner-indicator` | Internal visual slot that hosts the built-in shape or custom `icon`. |

## Composition

Use `variant` for built-in visuals: `ring`, `dots`, `bars`, or `pulse`. Use `icon` when a product
surface needs a custom SVG or brand-specific loading mark. Custom icons inherit the root size and
color and spin by default.

All parts accept standard `span` props through `Spinner`. Set `decorative` when the spinner is
paired with visible loading text, or pass a custom `label` when the default loading announcement
is not specific enough.

### Custom Icon

Pass `icon` to replace the built-in shape while keeping Spinner sizing and accessibility behavior.

<Preview>
  <SpinnerCustomIconExample />

  <Preview.Code>
    {`
      import { Spinner } from "moduix";

      export function SpinnerCustomIconDemo() {
        return (
          <Spinner
            size="lg"
            icon={
              <svg viewBox="0 0 24 24" fill="none" aria-hidden="true">
                <path
                  d="M12 3v4M12 17v4M3 12h4M17 12h4M5.64 5.64l2.83 2.83M15.53 15.53l2.83 2.83M18.36 5.64l-2.83 2.83M8.47 15.53l-2.83 2.83"
                  stroke="currentColor"
                  strokeWidth="2"
                  strokeLinecap="round"
                />
              </svg>
            }
          />
        );
      }
    `}

  </Preview.Code>
</Preview>
