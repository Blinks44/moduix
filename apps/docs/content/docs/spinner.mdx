## API Reference

`Spinner` is a moduix loading primitive and does not wrap a Base UI primitive.
Use it for pending controls, inline loading feedback, and compact empty states. By default it
announces loading with `role="status"`. Set `decorative` when nearby text already covers that.

## Anatomy

`Spinner` exposes one public root and one internal visual slot. The indicator renders either the
built-in glyph or the custom `icon` node.

```text
Spinner
└─ indicator
```

```tsx
<Spinner variant="ring" size="md" />
```

| Part                | Role                                                                 |
| ------------------- | -------------------------------------------------------------------- |
| `Spinner`           | Root status element. Use `variant`, `size`, `label`, and `icon`.     |
| `spinner-indicator` | Internal visual slot that hosts the built-in shape or custom `icon`. |

## Composition

Use `variant` for built-in visuals: `ring`, `dots`, `bars`, or `pulse`. Use `icon` to swap in a
custom SVG while keeping Spinner sizing, color, and default spin behavior.

`Spinner` forwards standard `span` props. Pass `label` for a more specific loading announcement.
Set `decorative` when the spinner is paired with visible loading text.

### Defaults

| Prop         | Default   | Values                          |
| ------------ | --------- | ------------------------------- |
| `variant`    | `ring`    | `ring`, `dots`, `bars`, `pulse` |
| `size`       | `md`      | `xs`, `sm`, `md`, `lg`, `xl`    |
| `label`      | `Loading` | Any React node                  |
| `decorative` | `false`   | `true`, `false`                 |
| `animated`   | `true`    | `true`, `false`                 |

### Custom Icon

Pass `icon` to replace the built-in shape without changing Spinner sizing or accessibility behavior.

  <SpinnerCustomIconExample />

      export function SpinnerCustomIconDemo() {
        return (
          <Spinner
            size="lg"
            icon={
              <svg viewBox="0 0 24 24" fill="none" aria-hidden="true">
                <path
                  d="M12 3v4M12 17v4M3 12h4M17 12h4M5.64 5.64l2.83 2.83M15.53 15.53l2.83 2.83M18.36 5.64l-2.83 2.83M8.47 15.53l-2.83 2.83"
                  stroke="currentColor"
                  strokeWidth="2"
                  strokeLinecap="round"
                />
              </svg>
            }
          />
        );
      }
    `}
