## Basic

Use `Listbox` for a visible collection where users pick one or more items without opening a popup.
Pass a `collection`, render `Content`, and pass each item through `Listbox.Item item={item}`.

<Preview>
  <ListboxExample />

  <Preview.Code>
    {`
      import { createListCollection } from "@ark-ui/react/collection";
      import { Listbox } from "@moduix/react";

      const countries = createListCollection({
        items: [
          { label: "United States", value: "us" },
          { label: "United Kingdom", value: "uk" },
          { label: "Canada", value: "ca" },
        ],
      });

      export function ListboxDemo() {
        return (
          <Listbox collection={countries}>
            <Listbox.Label>Select country</Listbox.Label>
            <Listbox.Content>
              {countries.items.map((item) => (
                <Listbox.Item key={item.value} item={item}>
                  <Listbox.ItemText>{item.label}</Listbox.ItemText>
                  <Listbox.ItemIndicator />
                </Listbox.Item>
              ))}
            </Listbox.Content>
          </Listbox>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`

## Anatomy

```text
Listbox / Listbox.Root
├─ Listbox.Label
├─ Listbox.Input
├─ Listbox.Content
│  ├─ Listbox.Empty
│  ├─ Listbox.ItemGroup
│  │  ├─ Listbox.ItemGroupLabel
│  │  └─ Listbox.Item[item]
│  │     ├─ Listbox.ItemText
│  │     └─ Listbox.ItemIndicator
│  └─ Listbox.Item[item]
│     ├─ Listbox.ItemText
│     └─ Listbox.ItemIndicator
└─ Listbox.ValueText
```

| Part                              | Role                                                          |
| --------------------------------- | ------------------------------------------------------------- |
| `Listbox` / `Listbox.Root`        | Owns collection, selected values, highlighted item, and mode. |
| `Listbox.RootProvider`            | Renders a listbox tree from state returned by `useListbox`.   |
| `Listbox.Label`                   | Accessible label for the listbox content.                     |
| `Listbox.Input`                   | Optional filter input connected to the same listbox state.    |
| `Listbox.Content`                 | Focusable listbox surface with keyboard navigation.           |
| `Listbox.Empty`                   | Empty-state content for filtered collections.                 |
| `Listbox.ItemGroup`               | Group wrapper for related collection items.                   |
| `Listbox.ItemGroupLabel`          | Label for a collection group.                                 |
| `Listbox.Item`                    | Selectable row. Pass the collection item through `item`.      |
| `Listbox.ItemText`                | Text used for display and accessibility.                      |
| `Listbox.ItemIndicator`           | Selected-state indicator with the moduix default check icon.  |
| `Listbox.ValueText`               | Selected value summary.                                       |
| `Listbox.Context` / `ItemContext` | Inline access to root and item state.                         |

## Composition

### Recommended composition

Render the Ark part tree directly: `Listbox` owns the collection, `Listbox.Content` owns focus and
keyboard navigation, and each `Listbox.Item` receives the original collection item through `item`.
Use `Listbox.ItemIndicator` when rows should reserve selected-state space.

### When to go custom

Use `asChild` only when a custom element can keep the required listbox semantics and receive Ark's
props. Use `ids` when listbox parts need stable IDs for composition with surrounding UI. Use
`useListbox` with `Listbox.RootProvider` when state must be controlled outside the rendered tree;
do not render `Listbox.Root` for the same state instance.

### Default props

| Prop                      | Default    | Notes                                                                 |
| ------------------------- | ---------- | --------------------------------------------------------------------- |
| `collection`              | -          | Required Ark collection. Use `createListCollection` or grid variants. |
| `defaultValue`            | `[]`       | Initial selected values for uncontrolled state.                       |
| `defaultHighlightedValue` | -          | Initial highlighted item for uncontrolled highlight state.            |
| `deselectable`            | -          | Follows Ark's empty-selection behavior for deselecting values.        |
| `disabled`                | `false`    | Disables the whole listbox.                                           |
| `disallowSelectAll`       | `false`    | Disables `Cmd/Ctrl + A` selection.                                    |
| `loopFocus`               | `false`    | Keeps arrow navigation from wrapping by default.                      |
| `orientation`             | `vertical` | Controls layout and arrow-key direction.                              |
| `selectionMode`           | `single`   | Supports `single`, `multiple`, and `extended`.                        |
| `selectOnHighlight`       | `false`    | Selects items as keyboard highlight changes.                          |
| `typeahead`               | `true`     | Enables typeahead unless a filtering input owns text entry.           |

### Behavioral notes

`value`, `defaultValue`, and `onValueChange(details)` use string arrays. Read `details.value` for
selected keys and `details.items` for selected collection items.

`highlightedValue`, `defaultHighlightedValue`, and `onHighlightChange(details)` control active
descendant movement. `onSelect(details)` fires for item selection with the selected value.

`Listbox.Input` is only needed for filtering. Set `typeahead={false}` when text input owns search
so typeahead and filtering do not compete.

Ark owns the ARIA listbox pattern, active-descendant focus management, keyboard navigation,
typeahead, Home/End behavior, disabled item handling, and orientation-specific arrow keys. Keep
`Listbox.Label` in the tree when the listbox needs a visible label.

## Examples

### Controlled

Control the selected string array from React state and update it from `details.value`.

<Preview>
  <ControlledListboxExample />

  <Preview.Code>
    {`
      import { useState } from "react";
      import { Listbox } from "@moduix/react";

      export function ControlledListboxDemo() {
        const [value, setValue] = useState(["md"]);

        return (
          <Listbox
            collection={sizes}
            value={value}
            onValueChange={(details) => setValue(details.value)}
          >
            <Listbox.Label>Select size</Listbox.Label>
            <Listbox.Content>
              {sizes.items.map((item) => (
                <Listbox.Item key={item.value} item={item}>
                  <Listbox.ItemText>{item.label}</Listbox.ItemText>
                  <Listbox.ItemIndicator />
                </Listbox.Item>
              ))}
            </Listbox.Content>
          </Listbox>
        );
      }
    `}

  </Preview.Code>

### Filtering

<Preview>

      import { Listbox } from "@moduix/react";

      export function FilteringListboxDemo() {
        const { collection, filter } = useListCollection({
          initialItems: frameworks,
          filter: (itemText, filterText) =>
