## API Reference

`Text` is the default typography primitive for body copy and supporting UI text. It stays focused on
three things:

- body typography defaults
- a small set of visual variants
- simple semantic composition

## Default Path

Use the default `p` for body copy. When only the intrinsic HTML tag should change, use `as`.

## Anatomy

`Text` renders a single root element.

```text
Text
└─ text or inline content
```

| Part   | Role                                                                |
| ------ | ------------------------------------------------------------------- |
| `Text` | Semantic text root with design-system typography, tone, and sizing. |

## Composition

`Text` has two composition paths:

| Pattern             | Best fit                                           |
| ------------------- | -------------------------------------------------- |
| default `p`         | Body copy and descriptive text                     |
| `as="small"`        | Common intrinsic tags with the same styling system |
| `render={<Link />}` | Router links and app-level custom primitives       |

Defaults stay intentionally small:

| Prop     | Default         | Values                                                 |
| -------- | --------------- | ------------------------------------------------------ |
| `as`     | `p`             | `p`, `span`, `small`, `strong`, `em`, `div`            |
| `render` | Unset           | React element or render function                       |
| `size`   | Render-based    | `xs`, `sm`, `md`, `lg`, `xl`                           |
| `weight` | Render-based    | `regular`, `medium`, `semibold`, `bold`                |
| `tone`   | `default`       | `default`, `muted`, `subtle`, `primary`, `destructive` |
| `align`  | Unset (`start`) | `left`, `center`, `right`                              |

Render-based defaults for `size` and `weight`:

- default root (`p`): `size="md"`, `weight="regular"`
- `as="small"` or `render={<small />}`: `size="sm"`, `weight="regular"`
- `as="strong"` or `render={<strong />}`: `size="md"`, `weight="semibold"`
- other intrinsic elements and custom components: `size="md"`, `weight="regular"`

## Custom Composition

### Custom Element

Use `render` when the root should be a custom component instead of an intrinsic tag.

<Preview>
  <TextCustomElementExample />

  <Preview.Code>
    {`
      import { Text } from "moduix";
      import type { ComponentPropsWithoutRef } from "react";

      type InlineLinkProps = ComponentPropsWithoutRef<"a">;

      function InlineLink(props: InlineLinkProps) {
        return <a {...props} />;
      }

      export function TextCustomElementDemo() {
        return (
          <Text render={<InlineLink href="/docs" />} tone="primary" weight="medium">
            Read the documentation
          </Text>
        );
      }
    `}

  </Preview.Code>
</Preview>
