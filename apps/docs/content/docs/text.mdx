## API Reference

`Text` is a typography primitive for body copy, inline text, and supporting descriptions.
Its API stays focused on semantic element choice, visual variants, and root `className`.

## Anatomy

`Text` is composed as a single typography root that renders the chosen semantic element.

```text
Text
└─ text or inline content
```

```tsx
<Text as="small" size="sm" tone="muted">
  Last updated just now
</Text>
```

| Part   | Role                                                                |
| ------ | ------------------------------------------------------------------- |
| `Text` | Semantic text root with design-system typography, tone, and sizing. |

## Composition

Use `as` for semantics and `size` for visual scale when they should differ. Built-in defaults
keep `strong` semibold and `small` at `sm`; other elements and custom components default to
regular `md` text. Use `className` and CSS variables for local styling.

### Defaults

| Prop     | Default         | Values                                                 |
| -------- | --------------- | ------------------------------------------------------ |
| `as`     | `p`             | Any valid React element or component                   |
| `size`   | Element-based   | `xs`, `sm`, `md`, `lg`, `xl`                           |
| `weight` | Element-based   | `regular`, `medium`, `semibold`, `bold`                |
| `tone`   | `default`       | `default`, `muted`, `subtle`, `primary`, `destructive` |
| `align`  | Unset (`start`) | `left`, `center`, `right`                              |

Element-based defaults for `size` and `weight`:

- `small`: `size="sm"`, `weight="regular"`
- `strong`: `size="md"`, `weight="semibold"`
- `p`, `span`, `em`, `div`, and custom components: `size="md"`, `weight="regular"`

## Examples

### Elements

Use `as` to choose the semantic element without leaving the typography scale.

  <TextElementsExample />

### Custom Element

Pass a React element or component to `as` when the rendered element belongs to a routing or
app-level primitive.

<Preview>
  <TextCustomElementExample />

  <Preview.Code>
    {`
      import { Text } from "moduix";
      import type { ComponentPropsWithoutRef } from "react";

      type InlineLinkProps = ComponentPropsWithoutRef<"a">;

      function InlineLink(props: InlineLinkProps) {
        return <a {...props} />;
      }

      export function TextCustomElementDemo() {
        return (
          <Text as={InlineLink} href="/docs" tone="primary" weight="medium">
            Read the documentation
          </Text>
        );
      }
    `}

  </Preview.Code>
</Preview>
