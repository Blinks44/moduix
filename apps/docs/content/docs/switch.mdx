This install path also pulls in the shared moduix foundation files that `Switch` depends on, such
as styles, icons, and small utilities.

`Switch` is a thin styled wrapper over the Base UI switch root. It keeps the default path short by
rendering `SwitchThumb` for you, while still exporting the thumb for explicit composition.

## Anatomy

Use `SwitchField` when the switch and label can live inside one wrapping `<label>`.

```text
SwitchField
├─ Switch
│  └─ SwitchThumb
└─ SwitchLabel
```

```tsx
<SwitchField>
  <Switch defaultChecked />
  <SwitchLabel>Enable notifications</SwitchLabel>
</SwitchField>
```

| Part          | Role                                                                 |
| ------------- | -------------------------------------------------------------------- |
| `Switch`      | Interactive root. Handles checked state, form props, and `size`.     |
| `SwitchThumb` | Optional thumb part for custom thumb content or primitive props.     |
| `SwitchField` | Optional wrapping label for the common inline field pattern.         |
| `SwitchLabel` | Optional text wrapper that inherits the switch label styling tokens. |

## Composition

Use `defaultChecked` for uncontrolled mode and `checked` with `onCheckedChange` for controlled
mode. `Switch` forwards the Base UI root props, so form props such as `name`, `value`,
`uncheckedValue`, `required`, `readOnly`, `form`, `inputRef`, `nativeButton`, and `render`
continue to work without extra wrapper API.

`size` is the only component-specific DX sugar. It scales the track and default thumb without
changing the composition model.

When the default thumb is enough, render `<Switch />` directly. When the thumb needs custom content
or custom primitive props, compose `SwitchThumb` explicitly through `children`.

For sibling-label layouts, switch the root to a native button:

```tsx
<div>
  <Switch id="product-updates" nativeButton render={<button />} />
  <label htmlFor="product-updates">Receive product updates</label>
</div>
```

Style the public parts with `className` and the `--switch-*` CSS variables.

### Defaults

| Prop        | Default    | Values/Notes                               |
| ----------- | ---------- | ------------------------------------------ |
| `size`      | `md`       | `xs`, `sm`, `md`, `lg`, `xl`               |
| `className` | undefined  | Root class name or state callback          |
| `children`  | auto thumb | Compose `SwitchThumb` manually when needed |

## Examples

### Controlled

Use controlled mode when the switch needs to coordinate with surrounding UI.

<Preview>
  <ControlledSwitchExample />

  <Preview.Code>
    {`
      import { Switch, SwitchField, SwitchLabel } from "moduix";
      import { useState } from "react";
      import styles from "./switch-demo.module.css";

      export function ControlledSwitchDemo() {
        const [checked, setChecked] = useState(true);

        return (
          <div className={styles.stack}>
            <SwitchField>
              <Switch checked={checked} onCheckedChange={setChecked} />
              <SwitchLabel>{checked ? "On" : "Off"}</SwitchLabel>
            </SwitchField>
            <span className={styles.hint}>Current value: {String(checked)}</span>
          </div>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .stack {
        display: flex;
        flex-direction: column;
        align-items: flex-start;
        gap: var(--spacing-3);
      }

      .hint {
        color: var(--color-muted-foreground);
        font-size: var(--text-xs);
        line-height: var(--line-height-text-xs);
      }
    `}

  </Preview.CSS>
</Preview>

### Custom Thumb

Compose `SwitchThumb` when the thumb needs custom content or separate styling hooks.

<Preview>
  <CustomIconSwitchExample />

  <Preview.Code>
    {`
      import { Switch, SwitchField, SwitchLabel, SwitchThumb } from "moduix";
      import type { ComponentProps } from "react";
      import styles from "./switch-demo.module.css";

      function PowerIcon(props: ComponentProps<"svg">) {
        return (
          <svg viewBox="0 0 16 16" fill="none" aria-hidden="true" focusable="false" {...props}>
            <path
              d="M8 2.5V7M5.1 4.3A5 5 0 1 0 10.9 4.3"
              stroke="currentColor"
              strokeWidth="1.8"
              strokeLinecap="round"
            />
          </svg>
        );
      }

      export function CustomIconSwitchDemo() {
        return (
          <SwitchField>
            <Switch defaultChecked>
              <SwitchThumb className={styles.customIconThumb}>
                <PowerIcon />
              </SwitchThumb>
            </Switch>
            <SwitchLabel>Use custom thumb icon</SwitchLabel>
          </SwitchField>
        );
      }
    `}

  </Preview.Code>

  <Preview.CSS>
    {`
      .customIconThumb {
        --switch-thumb-color-checked: var(--color-primary);
        --switch-thumb-color-unchecked: var(--color-muted-foreground);
        --switch-thumb-icon-size: 0.75rem;
      }
    `}
  </Preview.CSS>
</Preview>
