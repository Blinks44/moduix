Use `Separator` directly for non-interactive visual division between related pieces of content.
The default horizontal path already fills the available width, so the common case stays short.

## Anatomy

`Separator` exposes a single root slot with `data-orientation` for orientation-based styling.

```text
Separator
└─ root[data-orientation]
```

```tsx
<Separator orientation="horizontal" />
```

| Part        | Role                                                                                                   |
| ----------- | ------------------------------------------------------------------------------------------------------ |
| `Separator` | Root divider element. Controls `orientation`, accessibility attributes, styling, and element override. |

Style the root directly with `className` or the `--separator-*` CSS properties.

## Composition

`Separator` stays intentionally thin:

- one exported part
- orientation passed straight through to Base UI
- direct root styling with `className`
- a small `--separator-*` CSS contract for common visual tuning

Horizontal separators fill the container by default. Vertical separators are intended for inline
groups and default to `1em` height so they align with surrounding text.

Pass `orientation="vertical"` for inline layouts. Use `className` for direct styling,
or the small `--separator-*` CSS contract when you want to tune color, thickness, or length.

### Defaults

| Prop          | Default      | Values                                 |
| ------------- | ------------ | -------------------------------------- |
| `orientation` | `horizontal` | `horizontal`, `vertical`               |
| `className`   | `undefined`  | `string` or state-based class function |
| `style`       | `undefined`  | style object or state-based style fn   |

## Examples

### Vertical

Use `orientation="vertical"` for inline groups such as navigation actions.

  <VerticalSeparatorExample />

            <a href="#">Home</a>

### Styling

Use `className` or `--separator-*` custom properties when the divider needs a stronger visual
weight or a fixed length.
