## Install

<ShadcnInstall packageName="app-header" itemLabel="block" />

The command copies `app-header.tsx` and `app-header.module.css` into your configured components
directory, and installs `@moduix/react` and `lucide-react` when needed.

## Use in a route

Keep the header copy-owned: replace the navigation anchors, notification data, and menu actions with
your routes and application state. The block works independently or above the Sidebar dashboard.
