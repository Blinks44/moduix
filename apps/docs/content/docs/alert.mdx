## API Reference

`Alert` is a standalone moduix component and does not wrap a Base UI primitive.
Use it for inline messages that should stay in document flow. It supports icons,
controlled or uncontrolled dismissal, and optional close animation.

## Anatomy

`Alert` is composed from semantic content slots. `AlertIcon` is optional, and `AlertClose` is
rendered automatically when `withCloseButton` is set on the root.

```tsx

  <AlertIcon>
    <InfoIcon />
  </AlertIcon>
  <AlertContent>
    <AlertTitle>Workspace sync is active</AlertTitle>
    <AlertDescription>Changes are being synced across all connected devices.</AlertDescription>
  </AlertContent>

```

| Part               | Role                                                                 |
| ------------------ | -------------------------------------------------------------------- |
| `Alert`            | Root slot. Controls variant, size, visibility, and dismiss behavior. |
| `AlertIcon`        | Optional icon slot. Sizes child SVG icons consistently.              |
| `AlertContent`     | Text content wrapper for title and description.                      |
| `AlertTitle`       | Alert heading slot.                                                  |
| `AlertDescription` | Supporting message slot.                                             |
| `AlertClose`       | Close control wired to the nearest `Alert`.                          |

## Composition

Use `variant` for intent: `default`, `info`, `success`, `warning`, or `destructive`.
Use `open`, `defaultOpen`, and `onOpenChange` when dismissal must be coordinated with app state.
Set `withCloseButton` to render the built-in `CloseButton`; set `withDismissAnimation={false}` when
closing should remove the alert immediately. Pass `className` to the root or compound parts for
local styling. CSS variables should be overridden on the root alert slot.

### Defaults

| Prop                   | Default       | Values                                                               |
| ---------------------- | ------------- | -------------------------------------------------------------------- |
| `variant`              | `default`     | `default`, `info`, `success`, `warning`, `destructive`               |
| `size`                 | `md`          | `sm`, `md`, `lg`                                                     |
| `defaultOpen`          | `true`        | `true`, `false`                                                      |
| `withCloseButton`      | `false`       | `true`, `false`                                                      |
| `withDismissAnimation` | `true`        | `true`, `false`                                                      |
| `closeButtonLabel`     | `Close alert` | Any accessible text label for the built-in close button              |
| `role`                 | Auto          | `status` by default, `alert` when `variant="destructive"`, or custom |

### Variants

Use variants to separate neutral messages from informational, successful, warning, and destructive states.

<Preview>
  <AlertVariantsExample />

  <Preview.Code>
    {`
      import { Alert, AlertContent, AlertDescription, AlertIcon, AlertTitle, InfoIcon } from "moduix";

      const variants = ["default", "info", "success", "warning", "destructive"] as const;

      export function AlertVariantsDemo() {
        return (
          <div>
            {variants.map((variant) => (
              <Alert key={variant} variant={variant}>
                <AlertIcon>
                  <InfoIcon />
                </AlertIcon>
                <AlertContent>
                  <AlertTitle>{variant}</AlertTitle>
                  <AlertDescription>Use this alert for {variant} feedback.</AlertDescription>
                </AlertContent>
              </Alert>
            ))}
          </div>
        );
      }
    `}

  </Preview.Code>
</Preview>

### Dismissible

Set `withCloseButton` to render the standard close control. Use `open` and `onOpenChange` for
controlled dismissal.

<Preview>
  <AlertDismissibleExample />

  <Preview.Code>
    {`
      import { useState } from "react";
      import { Alert, AlertContent, AlertDescription, AlertIcon, AlertTitle, CheckFilledIcon } from "moduix";

      export function AlertDismissibleDemo() {
        const [open, setOpen] = useState(true);

        return (
          <Alert variant="success" open={open} onOpenChange={setOpen} withCloseButton>
            <AlertIcon>
              <CheckFilledIcon />
            </AlertIcon>
            <AlertContent>
              <AlertTitle>Saved</AlertTitle>
              <AlertDescription>The alert closes with the default exit animation.</AlertDescription>
            </AlertContent>
          </Alert>
        );
      }
    `}

  </Preview.Code>
</Preview>
